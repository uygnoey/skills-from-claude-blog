# この記事について

この記事は、プロダクトマネージャーが Claude、Claude Code、Claude Managed Agents を使って、初期の探索（リサーチ）から再利用可能なエージェント・ワークフローの出荷までを進める方法を紹介します。

# どんなときに役立つか

曖昧な調査を素早く具体的なプロトタイプに落とし込み、その後はクラウド上で長時間動くエージェント・セッションで継続的な運用作業を自動化したいときに役立ちます。

# 主なポイント

- ドキュメント上の設計だけでなく、実際の API で早期に動かして抽象化を圧力テストします。
- Claude/Claude Cowork はオープンエンドな探索に、Claude Code は ‘job to be done’ が明確になりエージェントとしてコード化できる段階で使います。
- PM 向けの例: 利用（Adoption）分析、開発者フィードバックのモニタリング、デモ作成。
- Managed Agents のセッションはクラウドで動作し、非同期の長時間タスクや実行間のメモリ蓄積が可能です。
