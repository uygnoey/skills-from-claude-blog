# PM-oriented Managed Agents: examples from the post

These are the three example agents listed in “Product development in the agentic era.”

## 1) Adoption analytics

Goal: query internal databases to surface outliers and patterns, using memory of prior runs to build on findings.

## 2) Developer sentiment monitoring

Goal: scan a specific list of domains for the latest developer feedback, fan out research in parallel, then synthesize common themes.

## 3) Demo building

Goal: use access to demo repositories, branding assets, and an event deck to produce a polished demo tailored to a target audience (conference or customer meeting).
