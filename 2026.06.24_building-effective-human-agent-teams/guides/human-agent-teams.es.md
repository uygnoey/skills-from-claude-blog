# Guía operativa para equipos humano–agente

## ¿De qué trata esta guía?
Una guía práctica para adoptar el trabajo en equipo humano–agente como un modelo operativo repetible.

## Cómo usarla
1. Elige un límite (frontera) de seguridad para el trabajo.
2. Dentro de ese límite, vuelve público y buscable el contexto clave.
3. Define un “roster” de humanos y agentes con propiedad clara y acceso a herramientas.
4. Establece una estrella polar.
5. Añade verificación y amplía la autonomía gradualmente.

## Lista de verificación
### Público por defecto dentro del límite
- Los canales son públicos por defecto (dentro del límite).
- Las decisiones se registran de forma consistente en canales/documentos/notas.
- La documentación se escribe para que los agentes puedan encontrarla y reutilizarla.

### Roles y herramientas
- Cada humano y agente tiene un alcance documentado.
- Se trabaja en hilos compartidos para reducir traspasos.
- El acceso a herramientas coincide con las responsabilidades del rol.

### Estrella polar
- La estrella polar está escrita, es ambiciosa y está vinculada a objetivos del negocio.
- Se nombran explícitamente los agentes que pueden proponer nuevos frentes de trabajo.
- Se protege el tiempo humano de alta calidad.

### Confianza y autonomía
- Al inicio, hay revisión manual.
- Existe un paso de verificación (verificador).
- Hay rúbricas para verificar entregables clave.
- La autonomía se amplía por tipo de tarea tras éxitos repetidos.
