## 이 글이 뭔가요
이 글은 Anthropic API Console의 **Workspaces** 기능을 소개합니다. Workspaces는 여러 Claude 배포를 더 쉽게 관리할 수 있도록 리소스와 권한/한도를 프로젝트 단위로 분리하는 환경입니다.

## 언제 유용한가요
스테이징/프로덕션처럼 환경을 분리하거나, 팀·프로젝트별로 접근 권한을 다르게 주고 싶거나, 배포 단위로 지출 한도와 속도 제한을 관리해야 할 때 유용합니다.

## 핵심 포인트
- Workspaces로 API 키, 사용량, 설정을 프로젝트 단위로 묶어 관리할 수 있습니다.
- 워크스페이스별 월간 지출 한도를 설정할 수 있습니다.
- 조직 한도 내에서 워크스페이스별 속도 제한을 조정할 수 있습니다.
- 워크스페이스 단위로 사용자 권한을 관리할 수 있습니다.
- 워크스페이스별 사용량과 비용을 분리해 모니터링할 수 있습니다.

## 번들 리소스
- Skill: Console Workspaces 생성/운영을 위한 간단한 실행 지침.
- Guide: 워크스페이스 범위 API 키, 한도, 접근 제어 설정 체크리스트.
