## この記事について
この記事は、Anthropic API Console の **Workspaces** を紹介する告知です。Workspaces は、複数の Claude デプロイをプロジェクト単位で整理し、権限や上限を分離して管理できる環境です。

## どんなときに役立つか
ステージング/本番の分離、チームやプロジェクトごとのアクセス制御、デプロイ単位での支出上限・レート制限の運用が必要なときに役立ちます。

## 主なポイント
- API キー、利用状況、設定などを Workspace 単位でまとめて管理できます。
- Workspace ごとの月次支出上限を設定できます。
- 組織の上限内で、Workspace ごとのレート制限を調整できます。
- Workspace レベルでユーザー権限を管理できます。
- Workspace 別に利用状況とコストを把握できます。
