# Checklist de operación de Workspaces (consola de la API de Anthropic)

## 1) Planificar
- Define qué representa un Workspace (proyecto, entorno, equipo o una combinación).
- Decide qué recursos deben separarse (claves de API, límites, control de acceso, reportes).

## 2) Crear
- Crea un Workspace en la consola de la API de Anthropic.
- Crea una clave de API con alcance de Workspace.

## 3) Controlar
- Establece un límite de gasto mensual por Workspace.
- Configura límites de tasa por Workspace (dentro de los límites de la organización).
- Asigna permisos de usuario a nivel de Workspace (mínimo privilegio).

## 4) Monitorear
- Revisa regularmente el uso y los costos por Workspace.
- Ajusta los límites de gasto y de tasa cuando cambien los patrones de tráfico.
