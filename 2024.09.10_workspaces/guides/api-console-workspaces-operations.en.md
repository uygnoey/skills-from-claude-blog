# Workspaces operations checklist (Anthropic API Console)

## 1) Plan
- Define what a Workspace represents (project, environment, team, or a combination).
- Decide which resources should be separated (API keys, limits, access control, reporting).

## 2) Create
- Create a Workspace in the Anthropic API Console.
- Create a workspace-scoped API key for that Workspace.

## 3) Control
- Set a monthly spend limit for the Workspace.
- Configure Workspace-specific rate limits (within org-wide limits).
- Assign user permissions at the Workspace level (least privilege).

## 4) Monitor
- Review usage and costs by Workspace regularly.
- Adjust spend and rate limits when traffic patterns change.
