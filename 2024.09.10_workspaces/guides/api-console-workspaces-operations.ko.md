# Workspaces 운영 체크리스트 (Anthropic API Console)

## 1) 계획
- Workspaces를 무엇으로 정의할지 정합니다(프로젝트, 환경, 팀, 또는 조합).
- 무엇을 분리할지 결정합니다(API 키, 한도, 접근 제어, 리포팅).

## 2) 생성
- Anthropic API Console에서 Workspace를 생성합니다.
- 해당 Workspace 범위의 API 키를 생성합니다.

## 3) 제어
- 워크스페이스 월간 지출 한도를 설정합니다.
- 조직 한도 내에서 워크스페이스별 속도 제한을 설정합니다.
- 워크스페이스 단위로 사용자 권한을 최소 권한 원칙으로 부여합니다.

## 4) 모니터링
- 워크스페이스별 사용량/비용을 주기적으로 확인합니다.
- 트래픽 패턴이 바뀌면 지출 한도·속도 제한을 조정합니다.
