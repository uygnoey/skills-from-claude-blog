# Checklist de implementación de Citations API

## Construcción
- Define las tareas (resúmenes, Q&A, soporte) y qué debe llevar cita.
- Proporciona documentos fuente (PDF o texto) y decide la estrategia de segmentación.
- Diseña la visualización de citas (marcadores en línea, notas al pie, salto al origen).

## Evaluación
- Mide recall y precisión de las citas.
- Controla la tasa de “no se encuentra en las fuentes” ante preguntas sin respaldo.
