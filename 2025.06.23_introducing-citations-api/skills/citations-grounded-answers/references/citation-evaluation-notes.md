# Citation evaluation notes

This reference summarizes key evaluation considerations mentioned or implied by the Citations announcement.

## What to measure
- **Recall**: do important claims have citations?
- **Precision**: do citations point to the right passages?
- **User trust**: do citations improve verification and reduce manual checking?

## Token considerations
The announcement notes standard token pricing, with additional input tokens used to process documents, and that quoted text returned for citations is not billed as output tokens.
