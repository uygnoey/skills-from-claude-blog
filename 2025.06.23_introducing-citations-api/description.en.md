## What is this post?
This post introduces Citations, an Anthropic API feature that lets Claude ground responses in your provided documents and return precise citations to the passages used.

## When is it useful?
Use this when users need verifiable outputs (summaries, Q&A, customer support) where every claim should be traceable to source text.

## Key points
- Citations links response claims to specific sentences/passages from source documents.
- Sources can be PDFs or plain text, and developers can also provide their own pre-chunked text.
- Citations is available on the Anthropic API and Google Cloud Vertex AI, with an update noting availability on Amazon Bedrock.

## Bundled resources
- Skill: a reusable workflow for building “grounded answers” experiences with citations.
- Guide: implementation checklist and product considerations (UX, pricing/tokens, evaluation).
