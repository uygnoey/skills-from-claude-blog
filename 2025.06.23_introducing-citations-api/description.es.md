## ¿De qué trata este post?
Este post presenta Citations, una función de la API de Anthropic que permite a Claude fundamentar las respuestas en documentos proporcionados y devolver citas precisas de los pasajes utilizados.

## ¿Cuándo es útil?
Es útil cuando los usuarios necesitan resultados verificables (resúmenes, preguntas y respuestas, soporte) y cada afirmación debe poder rastrearse hasta el texto fuente.

## Puntos clave
- Citations vincula las afirmaciones de la respuesta con frases/pasajes específicos de los documentos fuente.
- Las fuentes pueden ser PDF o texto plano, y los desarrolladores también pueden aportar su propio texto ya segmentado.
- Citations está disponible en la API de Anthropic y en Google Cloud Vertex AI, con una actualización que menciona su disponibilidad en Amazon Bedrock.

## Recursos incluidos
- Skill: flujo reutilizable para construir experiencias de “respuestas fundamentadas” con citas.
- Guide: checklist de implementación y consideraciones de producto (UX, precios/tokens, evaluación).
