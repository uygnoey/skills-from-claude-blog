## この記事について
この記事は、Anthropic API の Citations 機能を紹介します。Claude がユーザー提供の文書に基づいて回答を作成し、使用した箇所（文/パッセージ）への正確な引用を返せるようになります。

## どんなときに役立つか
要約、Q&A、カスタマーサポートなどで検証可能な出力が必要で、各主張がソース本文まで追跡できることが重要なときに役立ちます。

## 主なポイント
- Citations は、回答内の主張をソース文書の特定の文/パッセージに結び付けます。
- ソースは PDF またはプレーンテキストで、開発者が事前に分割したテキストを提供することもできます。
- Citations は Anthropic API と Google Cloud Vertex AI で利用でき、更新情報として Amazon Bedrock での提供も言及されています。

## 同梱リソース
- Skill: 引用付きの「根拠ある回答」体験を作るための再利用可能なワークフロー。
- Guide: 実装チェックリストとプロダクト観点（UX、価格/トークン、評価）。
