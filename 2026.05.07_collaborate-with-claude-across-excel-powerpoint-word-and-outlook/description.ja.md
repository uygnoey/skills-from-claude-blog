# Excel・PowerPoint・Word・Outlook 全体で Claude と協働する

## この記事について
本記事は、Excel・PowerPoint・Word 向け Claude が一般提供（GA）になり、Outlook 向け Claude が有料プラン向けのパブリックベータとして提供されることを案内します。

## どんなときに役立つか
Excel の分析、Word の文書作成、PowerPoint の資料作成、Outlook のメール仕分け／返信を行き来しながら、単一の会話コンテキストを Microsoft 365 アプリ間で維持したいときに役立ちます。

## 主なポイント
- 1つの会話で Excel・PowerPoint・Word・Outlook 間の文脈を引き継げます。
- 成果物間で変更を反映できます（例：Excel の前提を変更 → PowerPoint のグラフと Word のメモの数値に反映）。
- Outlook では受信トレイのトリアージ、返信下書きの作成、参加者の空き状況を確認した予定招待の作成などを支援します。
- 企業向けには Microsoft の管理機能で展開でき、可観測性（OpenTelemetry）や利用状況レポート（Analytics API）を設定できます。
