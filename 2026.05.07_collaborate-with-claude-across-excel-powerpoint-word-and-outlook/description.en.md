# Collaborate with Claude across Excel, PowerPoint, Word and Outlook

## What is this post?
This post announces that Claude for Excel, PowerPoint, and Word are generally available, and Claude for Outlook is in public beta for paid plans.

## When is it useful?
It’s useful when you need to move quickly between spreadsheet analysis, written narratives, presentations, and email triage/replies—while keeping one continuous conversation context across Microsoft 365 apps.

## Key points
- One conversation can carry context across Excel, PowerPoint, Word, and Outlook.
- Changes can propagate across artifacts (e.g., update an Excel assumption and reflect it in a PowerPoint chart and a Word memo).
- Outlook workflows include inbox triage, drafting replies as Outlook drafts, and creating calendar invites while checking attendee availability.
- Enterprises can deploy via Microsoft admin tooling, and may configure observability (OpenTelemetry) and usage reporting (Analytics API).

## Bundled resources
- Skill: A reusable workflow skill for cross-app collaboration in Microsoft 365.
- Guide: A short guide describing recommended workflows and admin considerations.
