# Excel, PowerPoint, Word, Outlook 전반에서 Claude와 협업하기

## 이 글이 뭔가요
이 글은 Excel, PowerPoint, Word용 Claude가 정식 출시(Generally Available)되었고, Outlook용 Claude는 유료 플랜 대상 공개 베타로 제공된다는 소식을 소개합니다.

## 언제 유용한가요
Excel 분석, Word 문서 작성, PowerPoint 발표자료 제작, Outlook 메일 분류/회신을 오가면서도 하나의 대화 맥락을 유지해야 할 때 유용합니다.

## 핵심 포인트
- 하나의 대화가 Excel, PowerPoint, Word, Outlook 전반에서 맥락을 유지할 수 있습니다.
- 산출물 간 변경이 자연스럽게 이어질 수 있습니다(예: Excel 가정 변경 → PowerPoint 차트와 Word 메모의 수치 반영).
- Outlook에서는 받은편지함 트리아지, 회신 초안 작성, 참석자 가능 시간을 확인하며 일정 초대 생성 등의 흐름을 지원합니다.
- 엔터프라이즈는 Microsoft 관리 도구를 통해 배포할 수 있으며, 관측 가능성(OpenTelemetry)과 사용량 리포팅(Analytics API) 설정을 지원합니다.

## 번들 리소스
- Skill: Microsoft 365 앱 간 협업 흐름을 재사용 가능한 방식으로 정리한 스킬.
- Guide: 권장 워크플로와 관리자 관점 배포/운영 포인트를 정리한 짧은 가이드.
