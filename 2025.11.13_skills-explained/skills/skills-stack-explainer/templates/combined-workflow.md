# Example: combining Projects, MCP, Skills, subagents, and prompts

When you ask: "Analyze how our top three competitors are positioning their new AI features and identify gaps we can exploit"

A possible flow:
1. Project context loads (uploaded research documents and project instructions).
2. MCP connections activate (e.g., Google Drive + GitHub + web search).
3. A competitive-analysis Skill provides the analytical framework.
4. Subagents execute in parallel (market research vs technical analysis).
5. Prompts refine the focus (e.g., "Focus on enterprise customers in healthcare").
