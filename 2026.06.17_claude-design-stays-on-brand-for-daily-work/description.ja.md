# Claude Design now stays on brand for daily work

## この記事について

Claude Designのアップデート告知で、デザインシステム準拠の強化、Claude Codeとの同期改善、キャンバス上での直接編集、出力/コネクタ先の拡充を紹介します。

## どんなときに役立つか

既存のデザインシステムを起点に、プロジェクト横断でブランドを保ちながら、デザインから実装（Claude Code）までつなぐワークフローが必要なときに役立ちます。

## 主なポイント

- GitHubリポジトリ/デザインファイル/アップロードからデザインシステムを取り込み、Claudeがコンポーネントで生成し、システムに照らして自動修正します。
- /design-syncでClaude Designに取り込み、Claude Codeのターミナルから /design で作成・編集・同期します。
- PDF/PowerPoint出力と、Adobe・Canva・Miro・Replit・Vercel・Wix などの接続先に送れます。
