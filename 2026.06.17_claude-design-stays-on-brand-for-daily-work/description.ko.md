# Claude Design now stays on brand for daily work

## 이 글이 뭔가요

Claude Design의 업데이트를 소개하는 글로, 디자인 시스템 준수 강화, Claude Code와의 동기화 개선, 캔버스에서 직접 편집, 더 많은 내보내기/연결 도구를 다룹니다.

## 언제 유용한가요

기존 디자인 시스템을 기반으로 ‘브랜드 일관성’을 유지하면서 디자인에서 코드 구현까지 이어지는 워크플로가 필요할 때 유용합니다. 특히 스크린샷에서 다시 시작하지 않고 Claude Code로 자연스럽게 넘기는 흐름에 초점이 있습니다.

## 핵심 포인트

- GitHub repo/디자인 파일/업로드에서 디자인 시스템을 가져오고, Claude가 컴포넌트로 제작한 뒤 시스템과 대조해 자동 수정합니다.
- /design-sync로 디자인 시스템을 Claude Design에 가져오고, 터미널의 Claude Code에서 /design로 디자인 프로젝트를 생성/편집/동기화합니다.
- PDF/PowerPoint 내보내기와 Adobe·Canva·Miro·Replit·Vercel·Wix 등 연결 앱 전송을 지원합니다.
