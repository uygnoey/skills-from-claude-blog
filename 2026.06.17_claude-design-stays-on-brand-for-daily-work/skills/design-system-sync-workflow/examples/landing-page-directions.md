# Daily landing-page exploration (example)

This example is based on the post’s suggested prompt: asking for multiple directions and then refining one.

## Prompt

Ask Claude Design for three directions for a next landing page that follow your imported design system.

## Review loop

1. Pick one direction worth refining.
2. Edit directly on the canvas for fine-grained adjustments.
3. Sync into Claude Code and continue implementation from the existing design work.
