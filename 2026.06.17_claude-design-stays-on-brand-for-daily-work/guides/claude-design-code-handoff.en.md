# Moving between Claude Design and Claude Code

## What this guide covers

A practical, daily-work workflow for staying on-brand in Claude Design and then moving into Claude Code without losing context.

## Workflow

1. Import your design system (repo/files/uploads).
2. Use /design-sync so Claude Design starts from your existing components.
3. Generate variations in Claude Design; edit directly on the canvas for precision.
4. When ready, hand off to Claude Code so it continues from the design artifact rather than starting from a screenshot.
5. Optionally export to PDF/PowerPoint or send to connected tools depending on your downstream needs.
