# Claude DesignからClaude Codeへつなぐ

## このガイドの範囲

Claude Designでブランド一貫性を保ちつつ、コンテキストを失わずにClaude Codeへ移行する日常業務向けワークフローをまとめます。

## ワークフロー

1. デザインシステムを取り込む（リポジトリ/ファイル/アップロード）。
2. /design-syncで、Claude Designが既存コンポーネントから始まるようにする。
3. Claude Designで案を出し、キャンバス上で直接微調整する。
4. 準備できたらClaude Codeへ引き継ぎ、スクリーンショット起点にならないようにする。
5. 必要に応じてPDF/PowerPoint出力や接続ツールへの送信を行う。
