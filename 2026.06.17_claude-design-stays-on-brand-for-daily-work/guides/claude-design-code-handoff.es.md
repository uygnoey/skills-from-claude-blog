# Pasar de Claude Design a Claude Code

## Qué cubre esta guía

Un flujo práctico para el trabajo diario: mantener la marca en Claude Design y luego continuar en Claude Code sin perder contexto.

## Flujo

1. Importa tu sistema de diseño (repo/archivos/subidas).
2. Usa /design-sync para que Claude Design parta de tus componentes existentes.
3. Genera variaciones en Claude Design y ajusta con edición directa en el lienzo.
4. Cuando esté listo, traspasa a Claude Code para continuar desde el artefacto de diseño (no desde una captura).
5. Según lo que necesites después, exporta a PDF/PowerPoint o envía a herramientas conectadas.
