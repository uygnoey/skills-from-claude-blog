# Claude Design과 Claude Code를 오가며 작업하기

## 이 가이드가 다루는 내용

Claude Design에서 브랜드 일관성을 유지한 뒤, 맥락을 잃지 않고 Claude Code로 이어가는 일상 업무용 워크플로를 정리합니다.

## 워크플로

1. 디자인 시스템을 가져옵니다(repo/파일/업로드).
2. /design-sync로 Claude Design이 기존 컴포넌트에서 시작하도록 설정합니다.
3. Claude Design에서 여러 방향을 만든 뒤, 캔버스에서 직접 미세 조정합니다.
4. 준비되면 Claude Code로 넘겨 스크린샷에서 다시 시작하지 않도록 합니다.
5. 필요에 따라 PDF/PowerPoint로 내보내거나 연결 도구로 전송합니다.
