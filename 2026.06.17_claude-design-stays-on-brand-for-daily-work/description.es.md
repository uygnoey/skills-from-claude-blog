# Claude Design now stays on brand for daily work

## ¿De qué trata este post?

Una actualización de producto sobre Claude Design: mejor adherencia al sistema de diseño, flujo más fluido con Claude Code, edición directa en el lienzo y más destinos de exportación/conectores.

## ¿Cuándo es útil?

Útil si quieres un flujo de trabajo de diseño a código que parta de tus sistemas de diseño existentes y mantenga consistencia entre proyectos, incluyendo traspaso a Claude Code sin empezar desde capturas de pantalla.

## Puntos clave

- Importa uno o varios sistemas de diseño (repo de GitHub, archivos de diseño o subidas) y deja que Claude construya con componentes y corrija contra el sistema.
- Usa /design-sync para traer el sistema a Claude Design y /design en Claude Code para crear/editar/sincronizar desde la terminal.
- Exporta a PDF/PowerPoint y envía a herramientas conectadas (Adobe, Canva, Miro, Replit, Vercel, Wix, etc.).

## Recursos incluidos

- Skill: flujo repetible para sincronizar sistema de diseño y traspaso diseño↔código ( skills/design-system-sync-workflow ).
- Guide: cómo moverse entre Claude Design y Claude Code ( guides/claude-design-code-handoff.*.md ).
