## ¿De qué trata este post?
Este post presenta **Agent Skills**: carpetas reutilizables con instrucciones y, opcionalmente, scripts y recursos que Claude puede cargar cuando son relevantes para una tarea.

## ¿Cuándo es útil?
Es útil cuando quieres que Claude siga procedimientos consistentes para trabajo especializado (por ejemplo: flujos de Excel, estándares de creación de documentos o guías de marca) en Claude apps, Claude Code y la API.

## Puntos clave
- Los Skills son **componibles**, **portátiles** y **eficientes** (Claude carga solo lo mínimo necesario), y pueden ser **potentes** al incluir código ejecutable.
- Claude detecta automáticamente cuándo un Skill es relevante.
- Los Skills se pueden gestionar en las apps de Claude, en Claude Console y vía la API (incluido el endpoint `/v1/skills`).
- En Claude Code, se pueden instalar Skills mediante plugins (por ejemplo, desde el marketplace anthropics/skills) o copiándolos a `~/.claude/skills`.
