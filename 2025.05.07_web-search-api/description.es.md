## ¿De qué trata este post?
Este post anuncia la herramienta Web Search para la Messages API de Anthropic, que permite a Claude buscar en la web y responder con citas.

## ¿Cuándo es útil?
Es útil cuando tu aplicación necesita información actualizada y verificable más allá del corte de entrenamiento (p. ej., cambios regulatorios, documentación reciente, noticias).

## Puntos clave
- Habilita la herramienta de web search en una solicitud a la Messages API.
- Claude decide cuándo conviene buscar, genera consultas dirigidas y puede realizar búsquedas progresivas (controladas por max_uses).
- Las respuestas incluyen citas de las fuentes recuperadas.
- Los administradores pueden controlar el uso con listas de dominios permitidos/bloqueados a nivel de organización.
- La sección de actualización menciona Web Fetch para obtener y analizar un URL específico.
