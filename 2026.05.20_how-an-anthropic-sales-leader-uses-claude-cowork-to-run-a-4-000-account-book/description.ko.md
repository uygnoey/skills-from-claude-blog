## 이 글이 뭔가요
이 글은 Anthropic의 미국 미드마켓 GTM 책임자인 Travis Bryant가 Claude Cowork를 사용해 4,000개 계정 규모의 영업 활동을 운영하는 방법(미팅 준비, 주간 포캐스트, 야간 계정 성향 점수화)을 공유합니다.

## 언제 유용한가요
여러 시스템에서 데이터를 모아 표준화된 브리프/리포트를 만들고, 최종 반영 전에 사람의 승인(human-in-the-loop)을 유지하는 반복 가능한 자동화 루틴이 필요한 영업 리더/영업운영 팀에 유용합니다.

## 핵심 포인트
- 일간: 일정 기반 미팅 준비로 (예: 회의실 예약) 같은 운영 작업을 보완하고, 통화 전에 고객 브리프를 만들어 둘 수 있습니다.
- 주간: Salesforce/BigQuery 등에서 데이터를 가져와 합의된 형식의 단일 페이지 웹 리포트를 배포하는 포캐스트 자동화가 가능합니다.
- 전략: 정의된 루브릭으로 수천 개 계정을 야간에 점수화한 뒤, 순위/근거를 담은 인터랙티브 대시보드를 만들 수 있습니다.
- 반복 개선: 테스트용 territory를 먼저 돌리고, 결과를 확인한 다음 가중치를 조정해 재실행합니다.

## 번들 리소스
- 글에서 설명한 “예약된 준비 / 주간 포캐스트 / 계정 스코어링” 패턴을 정리한 Agent Skill.
- 이 루틴들을 Claude Code 산출물로 옮기는 방법을 정리한 짧은 Guide.
