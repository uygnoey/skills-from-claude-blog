## ¿De qué trata este post?
Este post cuenta cómo Travis Bryant (Head of US Mid‑Market GTM en Anthropic) usa Claude Cowork para operar un portafolio de 4.000 cuentas, incluyendo preparación de reuniones, forecast semanal y un scoring nocturno de propensión por cuenta.

## ¿Cuándo es útil?
Es útil para líderes de ventas u operaciones que necesitan rutinas repetibles que reúnan datos de varios sistemas, generen briefs/reportes estandarizados y mantengan un paso de aprobación humana (human-in-the-loop).

## Puntos clave
- Diario: una tarea programada puede resolver logística (p. ej., reservar sala) y preparar un brief del cliente antes de cada llamada.
- Semanal: el forecast puede consolidarse extrayendo datos de sistemas como Salesforce y BigQuery y publicando un reporte web de una sola página en un formato acordado.
- Estratégico: una rutina nocturna puede puntuar miles de cuentas con una rúbrica definida y luego construir un panel interactivo con ranking y justificaciones.
- Iteración: ejecutar un territorio de prueba, revisar el resultado, ajustar pesos y volver a ejecutar.

## Recursos incluidos
- Una Agent Skill que captura los patrones reutilizables de “preparación programada / forecast semanal / scoring de cuentas” descritos en el post.
- Una guía breve para traducir estas rutinas a artefactos compatibles con Claude Code.
