## What is this post?
This post shares how Travis Bryant (Head of US Mid‑Market GTM at Anthropic) uses Claude Cowork to run day-to-day sales operations for a 4,000-account book, including meeting prep, weekly forecasting, and overnight account propensity scoring.

## When is it useful?
Useful for sales leaders or ops teams who need repeatable, automated routines that assemble data from multiple systems, produce standardized briefs/reports, and keep a human-in-the-loop approval step.

## Key points
- Daily: scheduled meeting prep can ensure logistics (e.g., room booking) and generate customer briefs ahead of calls.
- Weekly: an automated forecast rollup can pull data from systems like Salesforce and BigQuery, then publish a single-page web report in an agreed format.
- Strategic: an overnight routine can score thousands of accounts using a defined rubric, then produce an interactive dashboard with ranked accounts and rationales.
- Iteration loop: run a test territory, review the output, adjust weights, and re-run.
