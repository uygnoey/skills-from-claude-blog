# From “Cowork routines” to Claude Code artifacts

## What this guide covers
This guide summarizes the operational routines described in the post and maps them to Claude Code-friendly artifacts (skills, scheduled tasks, and outputs).

## Routine patterns described in the post
1. Scheduled meeting prep
2. Weekly forecast rollup
3. Overnight account propensity scoring

## How to package them
- Put reusable instructions into an **Agent Skill** (see `skills/sales-cowork-workflows/SKILL.md`).
- Keep outputs in formats your team already uses (the post mentions docs and single-page web reports).
- Maintain a human approval step before anything is sent or published.
