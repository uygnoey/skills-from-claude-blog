# De “rutinas de Cowork” a artefactos de Claude Code

## Qué cubre esta guía
Esta guía resume las rutinas operativas del post y las mapea a artefactos compatibles con Claude Code (skills, tareas programadas y outputs).

## Patrones de rutina descritos en el post
1. Preparación programada de reuniones
2. Consolidación semanal del forecast
3. Scoring nocturno de propensión por cuenta

## Cómo empaquetarlas
- Coloca instrucciones reutilizables en una **Agent Skill** (ver `skills/sales-cowork-workflows/SKILL.md`).
- Entrega resultados en formatos que el equipo ya usa (el post menciona documentos y reportes web de una sola página).
- Mantén un paso de aprobación humana antes de enviar o publicar.
