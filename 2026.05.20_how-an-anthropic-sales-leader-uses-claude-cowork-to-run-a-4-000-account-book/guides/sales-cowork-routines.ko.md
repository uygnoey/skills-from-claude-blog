# “Cowork 루틴”을 Claude Code 산출물로 옮기기

## 이 가이드가 다루는 내용
이 가이드는 글에서 소개된 운영 루틴을 요약하고, Claude Code에서 쓰기 좋은 산출물(스킬, 예약 작업, 출력물)로 매핑합니다.

## 글에서 설명한 루틴 패턴
1. 예약된 미팅 준비
2. 주간 포캐스트 롤업
3. 야간 계정 성향 점수화

## 패키징 방법
- 재사용 지침은 **Agent Skill**로 정리합니다(`skills/sales-cowork-workflows/SKILL.md`).
- 팀이 이미 쓰는 형식으로 결과를 내보냅니다(글에서는 문서 및 단일 페이지 웹 리포트를 언급).
- 발송/게시 전에는 사람의 승인 단계를 유지합니다.
