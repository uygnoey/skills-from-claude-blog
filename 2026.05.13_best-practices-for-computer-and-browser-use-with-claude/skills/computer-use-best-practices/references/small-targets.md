# Handling small targets (dense UIs)

Source: https://claude.com/blog/best-practices-for-computer-and-browser-use-with-claude

Recommendations from the post:
- Enable zoom for dense UIs.
- If you control the UI, increase target sizes (system DPI, browser zoom, UI scaling).
- Prefer keyboard alternatives (shortcuts, tab navigation) for very small elements.
- Consider source image resolution: 4K screenshots downscaled to 720p can make small UI elements extremely hard to click; for very high resolution displays, consider Opus 4.7 (higher image budget), lower DPI / display scaling, or focused screenshots.
