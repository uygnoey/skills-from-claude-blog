# Approaches tested that didn’t help (click accuracy)

Source: https://claude.com/blog/best-practices-for-computer-and-browser-use-with-claude

The post reports these did not produce reliable gains:
- Breaking the image into smaller tiles (sending quadrants/regions separately)
- Overlaying a coordinate grid on screenshots
- Choosing different resize algorithms (e.g., PIL LANCZOS vs. sips)
