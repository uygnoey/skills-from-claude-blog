## 概要
このガイドは、原文で説明されている Model Context Protocol（MCP）の要点を簡潔に整理します。

## 基本概念
- MCP は LLM を外部ツール／データに接続するためのオープン標準です。
- アシスタント側の MCP Client が、ツール／アプリ側の MCP Server に接続します。

## MCP が役立つ場面
- 複数の AI アシスタントと複数のツールを、標準的な方式でつなぎたいとき。
