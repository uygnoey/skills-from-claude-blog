## Visión general
Esta guía ofrece una orientación breve sobre Model Context Protocol (MCP) según el post fuente.

## Conceptos clave
- MCP es un estándar abierto para conectar LLMs con herramientas/datos externos.
- Los MCP Clients (lado del asistente) se conectan a MCP Servers (lado de la herramienta/app).

## Cuándo usar MCP
- Cuando quieres una integración estándar que funcione con múltiples asistentes y múltiples herramientas.
