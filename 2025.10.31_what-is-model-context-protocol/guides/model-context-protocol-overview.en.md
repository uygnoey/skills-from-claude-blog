## Overview
This guide provides a short orientation to Model Context Protocol (MCP) as described in the source post.

## Core concepts
- MCP is an open standard for connecting LLMs to tools/data.
- MCP Clients (assistant side) connect to MCP Servers (tool/app side).

## When to use MCP
- When you want one integration that can work across multiple AI assistants and multiple tools.
