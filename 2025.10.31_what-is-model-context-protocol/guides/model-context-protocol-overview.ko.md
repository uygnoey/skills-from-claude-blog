## 개요
이 가이드는 원문에서 설명하는 Model Context Protocol(MCP)의 핵심 개념을 짧게 정리합니다.

## 핵심 개념
- MCP는 LLM이 외부 도구/데이터에 연결하기 위한 오픈 표준입니다.
- 어시스턴트 측 MCP Client가 도구/앱 측 MCP Server에 연결합니다.

## MCP를 쓸 때
- 여러 AI 어시스턴트와 여러 도구를 표준 방식으로 연결하고 싶을 때(개별 통합 난립 방지).
