# Interactive connectors adoption playbook

## When to use interactive connectors
- You need preview-before-action (posting/sending).
- Output formatting matters (messages, decks).
- Users benefit from tuning parameters (analytics charts).

## Rollout steps
1. Pick one workflow and define approval rules.
2. Pilot with a small group.
3. Document edge cases and update prompts.
