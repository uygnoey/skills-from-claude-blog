## 이 글이 뭔가요
이 글은 MCP Apps를 통해 Claude에서 “인터랙티브 커넥터”를 제공한다는 내용을 소개합니다. 지원되는 도구가 채팅 안에서 라이브 UI로 표시되어, 탭을 전환하지 않고도 실행 내용을 확인·조정하며 협업할 수 있습니다.

## 언제 유용한가요
툴 액션을 더 안전하게 실행하고 싶을 때(게시/전송 전 미리보기), 매개변수를 대화 중에 빠르게 조정하며 반복하고 싶을 때, 작업 진행을 보면서 함께 의사결정하고 싶을 때 유용합니다.

## 핵심 포인트
- MCP Apps로 커넥터의 인터랙티브 UI를 Claude 대화 안에서 사용할 수 있습니다.
- Slack 메시지 초안의 포맷 미리보기, Asana 타임라인, Amplitude 차트, Figma/FigJam 다이어그램 등이 예시로 제시됩니다.
- MCP Apps는 Model Context Protocol(MCP)의 확장으로, 어떤 MCP 서버든 지원하는 AI 제품에서 인터랙티브 커넥터를 제공할 수 있게 합니다.

## 번들 리소스
- Skill: 기능 소개 글을 “평가 + 롤아웃” 체크리스트로 재사용 가능하게 정리하는 템플릿.
- Guide: 팀이 인터랙티브 커넥터 도입 여부를 판단할 때 쓰는 간단한 플레이북.
