## What is this post?
This post announces “interactive connectors” in Claude via MCP Apps, which let supported tools render a live UI inside the chat so you can review and collaborate on actions without switching tabs.

## When is it useful?
Use this when you want higher-confidence tool actions (preview before posting/sending), faster iteration (adjust parameters in-place), or richer collaboration (see what the tool is doing while you chat).

## Key points
- MCP Apps bring interactive UIs for connectors directly into Claude.
- Examples include Slack message drafting with a formatted preview, Asana timelines, Amplitude charts, and Figma/FigJam diagrams.
- MCP Apps are an extension to the Model Context Protocol (MCP) so any MCP server can deliver interactive connectors in supporting AI products.

- Guide: a short “adoption playbook” for teams deciding when to use interactive connectors.
