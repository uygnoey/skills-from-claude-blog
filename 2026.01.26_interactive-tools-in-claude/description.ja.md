## この記事について
この記事は、MCP Apps によって Claude に「インタラクティブ・コネクター」を導入することを発表しています。対応ツールがチャット内でライブ UI として表示され、タブ切り替えなしでアクションを確認・調整しながら共同作業できます。

## どんなときに役立つか
投稿/送信前にプレビューして安全に実行したいとき、会話の中でパラメータをすばやく調整して反復したいとき、ツールの動作を見ながら合意形成したいときに役立ちます。

## 主なポイント
- MCP Apps により、コネクターのインタラクティブ UI を Claude の会話内で利用できます。
- 例として、フォーマット済みプレビュー付きの Slack メッセージ作成、Asana のタイムライン、Amplitude のチャート、Figma/FigJam の図表化が挙げられています。
- MCP Apps は Model Context Protocol (MCP) の拡張であり、任意の MCP サーバーが対応 AI 製品上でインタラクティブ・コネクターを提供できます。

## 同梱リソース
- Skill: コネクター機能の告知を、評価と導入のチェックリストとして再利用できる形にまとめるテンプレート。
- Guide: インタラクティブ・コネクターの導入判断に使える短いプレイブック。
