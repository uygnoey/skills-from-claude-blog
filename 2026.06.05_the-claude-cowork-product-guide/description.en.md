# What is this post?
This post introduces Claude Cowork and outlines what makes it different from chat-style tools, plus the kinds of knowledge-work workflows it supports.

# When is it useful?
Use it when you want to delegate multi-step knowledge work that spans local files and connected apps, and you want a clear plan and a concrete deliverable.

# Key points
- Claude Cowork is positioned as a knowledge-work agent that can handle multi-step tasks end-to-end.
- It runs in the Claude desktop app and can read/write local files.
- It can work across connected apps (examples mentioned include Slack and Google Drive).
- It supports subagents, long-running work, and scheduled tasks.
- It emphasizes producing real deliverables with citations back to underlying files and messages.
