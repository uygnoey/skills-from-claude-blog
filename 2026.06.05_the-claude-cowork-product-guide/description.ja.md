# この記事について
この記事は Claude Cowork を紹介し、チャット型ツールとの違いや、対応するナレッジワークのワークフローの種類を概説します。

# どんなときに役立つか
ローカルファイルや接続アプリをまたぐ複数ステップのナレッジワークを委任し、明確な計画と具体的な成果物を得たいときに役立ちます。

# 主なポイント
- Claude Cowork は、複数ステップのタスクを最初から最後まで遂行できるナレッジワーク・エージェントとして位置づけられています。
- Claude のデスクトップアプリで動作し、ローカルファイルの読み書きができます。
- 接続されたアプリでも作業できます（例として Slack と Google Drive が挙げられています）。
- サブエージェント、長時間実行の作業、スケジュールされたタスクをサポートします。
- 実際の成果物を作り、根拠となるファイルやメッセージへの引用を重視します。
