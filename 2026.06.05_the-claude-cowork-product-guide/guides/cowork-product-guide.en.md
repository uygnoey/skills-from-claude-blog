# Claude Cowork product guide (summary)

## What it is
Claude Cowork is presented as a knowledge-work agent designed to carry multi-step tasks through to real deliverables.

## What makes it different
- Runs in the Claude desktop app.
- Can read and write local files.
- Can work across connected apps (examples mentioned include Slack and Google Drive).
- Supports subagents, long-running work, and scheduled tasks.
- Emphasizes citations back to underlying files and messages.

## How to get started (high-level)
1. State the goal.
2. Define the desired outcome (deliverable format, scope, and constraints).
3. If relevant, define the cadence.
4. Let Cowork propose a plan, then iterate.
