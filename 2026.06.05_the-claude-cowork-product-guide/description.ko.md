# 이 글이 뭔가요
이 글은 Claude Cowork를 소개하고, 채팅형 도구와의 차이점 및 지원하는 지식 작업(knowledge work) 워크플로를 개괄합니다.

# 언제 유용한가요
로컬 파일과 연결된 앱을 넘나드는 다단계 지식 작업을 위임하고, 명확한 계획과 구체적인 산출물을 얻고 싶을 때 유용합니다.

# 핵심 포인트
- Claude Cowork는 다단계 작업을 처음부터 끝까지 수행하는 지식 작업 에이전트로 소개됩니다.
- Claude 데스크톱 앱에서 실행되며 로컬 파일을 읽고/쓸 수 있습니다.
- 연결된 앱 전반에서 작업할 수 있습니다(예: Slack, Google Drive).
- 서브에이전트, 장시간 실행 작업, 스케줄드 태스크를 지원합니다.
- 실제 산출물과 함께, 근거가 되는 파일/메시지로의 인용을 강조합니다.
