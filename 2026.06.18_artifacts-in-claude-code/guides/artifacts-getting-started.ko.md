# Claude Code 아티팩트로 진행 상황을 ‘공유 가능한 라이브 페이지’로 만들기

## 아티팩트란?

아티팩트는 Claude Code 세션의 전체 컨텍스트(코드베이스, 커넥터, 대화)를 바탕으로 생성되는 실시간·인터랙티브 웹 페이지입니다.

## 언제 쓰면 좋나요

진행 중인 작업을 팀과 공유할 때, 세션 전체를 다시 설명하지 않고도 한 번의 링크로 현재 상태를 보여주고 싶다면 유용합니다.

PR 워크스루, 시스템 설명, 대시보드, 릴리스 체크리스트 같은 형태가 대표적입니다.

## 사용 흐름

1. 아티팩트를 만들어 달라고 요청합니다(또는 ‘시각적인 결과’를 요청합니다).
2. 첫 버전을 확인한 뒤 작업을 계속합니다.
3. 새로 알게 된 내용을 반영해 재발행합니다. 페이지는 제자리에서 갱신되고, 링크는 동일하며 버전 히스토리가 남습니다.
4. 페이지 헤더의 링크를 팀에 공유합니다.

## 프라이버시/공유

아티팩트는 기본적으로 작성자에게 비공개이며, 조직 내에서만 공유할 수 있고 공개로 전환할 수 없습니다.
