# Using Claude Code artifacts for shareable, live status

## What are artifacts?

Artifacts are live, interactive web pages generated from the full context of your Claude Code session (codebase, connectors, and conversation).

## When to use them

Use artifacts when you want to share in-progress work as a single link that teammates can open and explore, without having to replay the whole session.

Common formats include PR walkthroughs, system explainers, dashboards, and release checklists.

## Workflow

1. Ask for an artifact (or ask for something visual).
2. Review the first version, then keep working.
3. Republish as you learn more; the open page refreshes and the link stays the same with version history.
4. Share the link from the page header with teammates.

## Privacy and sharing

Artifacts are private to the author by default, shareable within your organization, and cannot be made public.
