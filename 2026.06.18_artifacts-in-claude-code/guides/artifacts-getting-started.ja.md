# Claude Code artifacts で進捗を共有できるライブページにする

## artifacts とは

artifacts は、Claude Code セッションのフルコンテキスト（コードベース、コネクタ、会話）から生成されるライブで対話的な Web ページです。

## どんなときに役立つか

進行中の作業を、チームが単一リンクで開いて探索できる形で共有したいときに役立ちます。セッション全体を説明し直す必要がありません。

代表的な形式は PR のウォークスルー、システム説明、ダッシュボード、リリースチェックリストです。

## 使い方の流れ

1. artifact を依頼します（または「視覚的なものが欲しい」と依頼します）。
2. 最初のバージョンを確認して作業を続けます。
3. 学びに合わせて再公開します。ページはその場で更新され、リンクは同じでバージョン履歴が残ります。
4. ページヘッダーからリンクを共有します。

## プライバシーと共有

artifacts はデフォルトで作者のみが閲覧でき、組織内で共有できますが公開はできません。
