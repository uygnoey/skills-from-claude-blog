# Usar artifacts de Claude Code para compartir el estado en vivo

## ¿Qué son los artifacts?

Los artifacts son páginas web en vivo e interactivas generadas a partir del contexto completo de tu sesión de Claude Code (código base, conectores y conversación).

## Cuándo usarlos

Úsalos cuando quieras compartir trabajo en curso como un enlace único que el equipo pueda abrir y explorar sin tener que repasar toda la sesión.

Formatos comunes: walkthroughs de PR, explicaciones de sistemas, dashboards y checklists de release.

## Flujo de trabajo

1. Pide un artifact (o pide algo visual).
2. Revisa la primera versión y continúa trabajando.
3. Vuelve a publicar a medida que avances; la página se actualiza en el lugar y el enlace se mantiene con historial de versiones.
4. Comparte el enlace desde el encabezado de la página.

## Privacidad y compartición

Los artifacts son privados por defecto, se comparten solo dentro de la organización y no pueden hacerse públicos.
