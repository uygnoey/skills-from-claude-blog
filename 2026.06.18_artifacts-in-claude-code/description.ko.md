# Claude Code now supports artifacts

## 이 글이 뭔가요

이 글은 Claude Code 아티팩트(artifacts)를 소개합니다. 아티팩트는 Claude Code 세션의 작업 진행 상황을 바탕으로 만들어지는 실시간·인터랙티브 웹 페이지입니다.

## 언제 유용한가요

디버깅, PR 리뷰, 아키텍처 파악, 대시보드, 릴리스 체크리스트 등 진행 중인 작업을 팀과 공유해야 할 때, 한 번의 링크로 보고/공유할 수 있는 뷰가 필요하면 유용합니다.

## 핵심 포인트

- 아티팩트는 세션의 전체 컨텍스트(코드베이스, 커넥터, 대화)를 사용해 생성됩니다.
- 세션이 진행되며 아티팩트가 제자리에서 업데이트되고, 동일 링크에 새 버전이 발행되며 버전 히스토리를 제공합니다.
- 기본적으로 작성자에게 비공개이며, 조직 내 인증된 멤버에게 공유할 수 있고 공개로 전환할 수 없습니다.
- 갤러리에서 생성한 아티팩트를 찾아보고 관리할 수 있습니다.
- 역할별 예시 프롬프트(법무/오픈소스, 프라이버시, 보안, FinOps, 엔지니어, 디자이너/프런트엔드, 스태프/아키텍트, SRE/온콜, 엔지니어링 매니저)가 제공됩니다.

## 번들 리소스

- 아티팩트 요청을 위한 역할별 예시 프롬프트
