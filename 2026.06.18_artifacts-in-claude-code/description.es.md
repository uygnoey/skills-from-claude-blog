# Claude Code now supports artifacts

## ¿De qué trata este post?

Este post anuncia los artifacts de Claude Code: páginas web en vivo e interactivas que resumen y visualizan el trabajo en curso a partir del contexto completo de una sesión.

## ¿Cuándo es útil?

Son útiles cuando necesitas compartir el estado actual del trabajo (depuración, revisión de PR, exploración de arquitectura, dashboards o seguimiento de releases) mediante un enlace único que se actualiza a medida que avanzas.

## Puntos clave

- Los artifacts se construyen con el contexto completo de la sesión (código base, conectores y conversación).
- Se actualizan en el lugar a medida que progresa la sesión; cada publicación crea una nueva versión en el mismo enlace con historial de versiones.
- Son privados por defecto, se pueden compartir dentro de la organización y no pueden hacerse públicos.
- Una galería permite explorar y administrar tus artifacts.
- Incluye prompts de ejemplo por rol (Legal/OSS, Privacidad, Seguridad, FinOps, Ingeniería, Diseño, Staff/Arquitectura, SRE y management).
