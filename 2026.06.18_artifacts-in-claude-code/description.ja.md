# Claude Code now supports artifacts

## この記事について

この記事は、Claude Code の artifacts（アーティファクト）を紹介します。artifacts は、セッションの進行状況をもとに作られるライブで対話的な Web ページです。

## どんなときに役立つか

デバッグ、PR レビュー、アーキテクチャ把握、ダッシュボード、リリースチェックリストなどの進捗を、作業しながら更新される単一リンクとして共有したいときに役立ちます。

## 主なポイント

- artifacts はセッションのフルコンテキスト（コードベース、コネクタ、会話）から生成されます。
- セッションの進行に合わせてページがその場で更新され、同じリンクに新しいバージョンが公開され、履歴から復元できます。
- デフォルトでは作者のみが閲覧でき、組織内で共有可能ですが公開はできません。
- ギャラリーで artifacts を閲覧・管理できます。
- 複数の職種向けの例示プロンプト（法務/OSS、プライバシー、セキュリティ、FinOps、エンジニア、デザイン/フロントエンド、スタッフ/アーキテクト、SRE/オンコール、EM）が含まれます。
