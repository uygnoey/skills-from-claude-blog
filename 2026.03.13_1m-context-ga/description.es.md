## ¿De qué trata este post?
Este post anuncia que Claude Opus 4.6 y Claude Sonnet 4.6 ahora admiten una ventana de contexto completa de 1M de tokens con disponibilidad general (GA) en Claude Platform.

## ¿Cuándo es útil?
Es útil cuando necesitas analizar o referenciar entradas muy grandes en una sola solicitud (por ejemplo, bases de código extensas, documentos legales largos, diffs grandes, trazas de agentes de larga duración, PDFs grandes, conjuntos de datos y muchas imágenes) sin tener que diseñar soluciones para límites de contexto más cortos.

## Puntos clave
- El contexto de 1M está en GA para Opus 4.6 y Sonnet 4.6.
- Se aplica el precio estándar a toda la ventana de 1M (sin recargo por contexto largo).
- Los límites de tasa estándar se aplican en toda la ventana.
- El límite de medios aumenta hasta 600 imágenes o 600 páginas de PDF por solicitud (antes 100).
- Las solicitudes de más de 200K tokens funcionan sin encabezado beta; si ya lo envías, se ignora.
- El contexto de 1M también está disponible en Claude Code para usuarios Max, Team y Enterprise con Opus 4.6.
