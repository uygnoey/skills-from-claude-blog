# Getting started with Claude Cowork (best practices)

## What this guide covers
How to decide whether a task is a good fit for Claude Cowork, and how to frame the task so Cowork can produce a useful deliverable.

## When to use Cowork vs chat
Use chat when you want a short interaction (question, explanation, brainstorm, gut check).

Use Claude Cowork when you want a deliverable and/or the task:
- is multi-step
- touches more than one file or file type
- touches more than one app
- is recurring
- is better described as a task than a question

## Quick checklist: is this task Cowork-shaped?
A task is a good fit when:
1. More than one thing goes in.
2. A file comes out.
3. You’ll do it again.
4. You already know what good looks like.
5. The middle is the boring part.

## How to get started
1. Open the Claude desktop app and switch to the Claude Cowork tab.
2. Give Cowork something to work with (files, a folder, or connected apps).
3. Tell Cowork the outcome you want (describe the deliverable).
4. Provide necessary context.
5. Ask Cowork to repeat your ask back and ask clarifying questions before it starts.

Suggested prompt:
“Before we begin, repeat my ask back to me so we're aligned, then ask me as many clarifying questions as you have.”
