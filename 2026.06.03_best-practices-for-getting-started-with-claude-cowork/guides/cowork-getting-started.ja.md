# Claude Cowork を始める（ベストプラクティス）

## このガイドの内容
どのタスクが Claude Cowork に適しているかを判断する方法と、Cowork が有用な成果物を作れるよう依頼を組み立てる方法をまとめます。

## Cowork とチャットの使い分け
質問、説明、ブレスト、軽い壁打ちなど短いやり取りならチャット。

成果物が必要で、かつ／または次の条件に当てはまるなら Claude Cowork。
- 複数ステップの作業
- 複数ファイル／複数のファイル形式に触れる
- 複数アプリにまたがる
- 繰り返し発生する
- 「質問」より「タスク」として説明したほうが自然

## クイックチェック: Cowork 向きのタスクか？
次に当てはまるとき、Cowork に適しています。
1. 入力が複数ある
2. ファイルが出力される
3. またやる
4. 良い結果のイメージがすでにある
5. 途中が退屈（抽出、集計、突合、再フォーマットなど）

## 始め方
1. Claude デスクトップアプリを開き、Claude Cowork タブに切り替える。
2. Cowork に作業対象を渡す（ファイル、フォルダ、または接続済みアプリ）。
3. 欲しい結果（成果物）を伝える。
4. 必要な文脈を渡す。
5. 開始前に、依頼内容の復唱と確認質問をするよう指示する。

推奨プロンプト:
“Before we begin, repeat my ask back to me so we're aligned, then ask me as many clarifying questions as you have.”
