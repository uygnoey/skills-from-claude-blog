# Claude Cowork 시작하기 (모범 사례)

## 이 가이드에서 다루는 내용
어떤 작업이 Claude Cowork에 적합한지 판단하는 방법과, Cowork가 유용한 산출물을 만들 수 있도록 작업을 어떻게 요청할지 정리합니다.

## Cowork vs 채팅: 언제 무엇을 쓸까
질문, 설명, 브레인스토밍, 빠른 검토처럼 짧은 상호작용이면 채팅을 권합니다.

산출물이 필요하거나, 다음 조건에 해당하면 Claude Cowork가 적합합니다.
- 여러 단계로 이루어진 작업
- 둘 이상의 파일/파일 형식을 다룸
- 둘 이상의 앱을 다룸
- 반복되는 작업
- ‘질문’보다 ‘작업’에 가깝게 설명되는 일

## 빠른 체크: 이 작업은 Cowork에 맞나요?
다음에 해당하면 Cowork에 잘 맞습니다.
1. 입력이 여러 가지다.
2. 파일이 출력된다.
3. 다시 할 가능성이 높다.
4. 좋은 결과가 무엇인지 이미 알고 있다.
5. 중간 과정이 지루하다(추출, 정리, 대조, 재포맷 등).

## 시작 방법
1. Claude 데스크톱 앱을 열고 Claude Cowork 탭으로 전환합니다.
2. Cowork가 작업할 대상을 제공합니다(파일, 폴더, 또는 연결된 앱).
3. 원하는 결과(산출물)를 명확히 말합니다.
4. 필요한 맥락을 제공합니다.
5. 시작 전 요청을 다시 말하고 확인 질문을 하도록 지시합니다.

권장 프롬프트:
“Before we begin, repeat my ask back to me so we're aligned, then ask me as many clarifying questions as you have.”
