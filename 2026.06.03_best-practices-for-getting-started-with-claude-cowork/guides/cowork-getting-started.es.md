# Empezar con Claude Cowork (buenas prácticas)

## Qué cubre esta guía
Cómo decidir si una tarea encaja bien en Claude Cowork y cómo plantearla para que Cowork produzca un entregable útil.

## Cuándo usar Cowork vs chat
Usa chat cuando quieras una interacción breve (pregunta, explicación, lluvia de ideas, validación rápida).

Usa Claude Cowork cuando necesites un entregable y/o la tarea:
- sea de varios pasos
- afecte a más de un archivo o tipo de archivo
- afecte a más de una app
- sea recurrente
- se describa mejor como una tarea que como una pregunta

## Checklist rápido: ¿esta tarea es adecuada para Cowork?
Una tarea encaja bien cuando:
1. Entran varias cosas.
2. Sale un archivo.
3. La harás de nuevo.
4. Ya sabes cómo se ve un buen resultado.
5. La parte intermedia es la aburrida (extraer, compilar, conciliar, reformatear).

## Cómo empezar
1. Abre la app de escritorio de Claude y cambia a la pestaña Claude Cowork.
2. Dale a Cowork algo con lo que trabajar (archivos, una carpeta o apps conectadas).
3. Dile el resultado que quieres (describe el entregable).
4. Proporciona el contexto necesario.
5. Pídele que repita tu solicitud y haga preguntas de aclaración antes de comenzar.

Prompt sugerido:
“Before we begin, repeat my ask back to me so we're aligned, then ask me as many clarifying questions as you have.”
