# Example: running three tasks in parallel

1. Open agent view:

```bash
claude agents
```

2. Dispatch three independent tasks as background sessions (type each prompt and press Enter):

- "Investigate the flaky SettingsChangeDetector test"
- "Review PR 1234 and suggest fixes"
- "Scan recent logs for auth timeouts"

3. Watch for sessions that need input (blocked), and use `Space` to peek.

4. Reply in the peek panel, or attach (`Enter`) when you need the full transcript.

5. When everything is complete, stop and delete sessions you no longer need.
