# Code w/ Claude London 2026: Rethinking how we build

## What is this post?
An event recap for Code w/ Claude London 2026, with links to session recordings and a short overview of product updates discussed during the event.

## When is it useful?
Useful if you want a high-level overview of the new Claude Platform capabilities mentioned at the event—especially private execution options for Claude Managed Agents—and pointers to where to learn more.

## Key points
- Claude Managed Agents can now operate in a sandbox you control and connect to private Model Context Protocol (MCP) servers. 
- Self-hosted sandboxes are in public beta and let agent execution run on infrastructure you control (including some managed providers), while orchestration remains on Anthropic infrastructure.
- MCP tunnels are in research preview and provide an outbound-only, encrypted way to reach private MCP servers without exposing inbound endpoints.
