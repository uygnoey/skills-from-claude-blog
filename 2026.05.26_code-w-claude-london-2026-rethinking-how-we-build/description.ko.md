# Code w/ Claude London 2026: Rethinking how we build

## 이 글이 뭔가요
Code w/ Claude London 2026 행사 리캡으로, 세션 녹화 링크와 함께 행사에서 언급된 제품 업데이트를 간단히 요약합니다.

## 언제 유용한가요
행사에서 언급된 Claude Platform의 신규 기능을 큰 그림으로 파악하고(특히 Claude Managed Agents의 프라이빗 실행 옵션), 추가 학습 경로를 빠르게 찾고 싶을 때 유용합니다.

## 핵심 포인트
- Claude Managed Agents가 이제 사용자가 제어하는 샌드박스에서 실행될 수 있으며, 프라이빗 Model Context Protocol(MCP) 서버에 연결할 수 있습니다.
- Self-hosted sandbox는 public beta로 제공되며, 에이전트 실행을 사용자의 인프라(일부 매니지드 제공자 포함)에서 수행하게 하되 오케스트레이션은 Anthropic 인프라에 남습니다.
- MCP tunnel은 research preview로 제공되며, 인바운드 엔드포인트를 공개하지 않고도 프라이빗 MCP 서버에 아웃바운드 전용 암호화 연결을 제공하는 방식입니다.
