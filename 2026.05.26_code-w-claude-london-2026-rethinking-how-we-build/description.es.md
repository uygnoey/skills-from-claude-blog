# Code w/ Claude London 2026: Rethinking how we build

## ¿De qué trata este post?
Un resumen del evento Code w/ Claude London 2026, con enlaces a las grabaciones de las sesiones y una breve descripción de las novedades de producto mencionadas durante el evento.

## ¿Cuándo es útil?
Es útil si quieres una visión general de las nuevas capacidades de la Claude Platform mencionadas en el evento—en especial, opciones de ejecución privada para Claude Managed Agents—y enlaces para profundizar.

## Puntos clave
- Claude Managed Agents ahora puede ejecutarse en un sandbox que tú controlas y conectarse a servidores privados de Model Context Protocol (MCP).
- Los sandboxes autoalojados están en beta pública y permiten ejecutar el agente en infraestructura bajo tu control (incluidos algunos proveedores gestionados), mientras que la orquestación permanece en la infraestructura de Anthropic.
- Los túneles MCP están en “research preview” y proporcionan una conexión cifrada solo saliente para llegar a servidores MCP privados sin exponer endpoints entrantes.
