# Code w/ Claude London 2026: Rethinking how we build

## この記事について
Code w/ Claude London 2026 のイベントまとめ記事で、セッション録画へのリンクと、イベントで触れられたプロダクト更新の要点が簡潔に紹介されています。

## どんなときに役立つか
イベントで言及された Claude Platform の新機能（特に Claude Managed Agents のプライベート実行オプション）を俯瞰し、追加の学習先に素早く辿り着きたいときに役立ちます。

## 主なポイント
- Claude Managed Agents は、ユーザーが制御するサンドボックスで動作でき、プライベートな Model Context Protocol（MCP）サーバーに接続できます。
- Self-hosted sandbox はパブリックベータで提供され、エージェントの実行を自分のインフラ（いくつかのマネージド提供者を含む）で行いつつ、オーケストレーションは Anthropic のインフラに残ります。
- MCP トンネルは research preview として提供され、インバウンドのエンドポイントを公開せずに、プライベートな MCP サーバーへアウトバウンド専用の暗号化接続を提供します。
