## 이 글이 뭔가요
코드를 한 줄도 써본 적 없던 프로젝트 매니저 Kostiantyn Vlasenko가 Claude Code를 중심으로 *Respiro*라는 iOS 스트레스 관리 앱을 만들고, 6주 이내에 App Store에 출시한 사례를 다룹니다. 출처: https://claude.com/blog/how-a-non-technical-project-manager-built-and-shipped-a-stress-management-app-with-claude-code-in-six-weeks

## 언제 유용한가요
- Claude Code를 주 개발 환경으로 사용해 실제 앱을 처음부터 끝까지 출시하는 흐름을 참고하고 싶을 때
- 아키텍처/구현/리뷰 역할을 분리한 멀티 에이전트(서브에이전트) 접근을 고민할 때

## 핵심 포인트
- Built With Opus 4.6 Claude Code Hackathon(2026년 2월)을 위해 약 72시간 안에 초기 프로토타입을 만들었고, 전체적으로는 6주 이내에 App Store 출시까지 진행했습니다.
- TCA 아키텍트, Swift 개발자, Metal 전문가, 코드 리뷰어 등 15개 이상의 전문 서브에이전트를 작은 팀처럼 조율했다고 설명합니다.
- 코딩 외에도 스크린샷 기반 디버깅, App Store 제출, 마케팅/그로스 작업에도 Claude Code를 활용했습니다.

## 번들 리소스
- 이 폴더에는 글에서 소개된 작업 흐름을 기반으로 한 재사용 가능한 스킬/가이드와, “Day Zero Questionnaire” 및 간단한 프로젝트 브리프 템플릿을 포함합니다.
