# Publicar una app real con Claude Code (checklist basada en un caso)

Fuente: https://claude.com/blog/how-a-non-technical-project-manager-built-and-shipped-a-stress-management-app-with-claude-code-in-six-weeks

## Qué cubre esta guía
Una checklist práctica derivada del post, centrada en: prototipo → especialización con subagentes → endurecimiento para producción → publicación en App Store → medición.

## Checklist
1. **Define un objetivo en una frase** y enumera las restricciones (plataforma, dispositivos, tiempo).
2. **Prototipa rápido** con un flujo end-to-end que puedas desechar.
3. **Asigna roles especializados** (arquitectura, implementación, revisión, expertos de plataforma).
4. **Endurece para producción** (reportes de fallos, analítica, depuración con capturas de pantalla).
5. **Prepara el lanzamiento** (materiales de envío, copy de la tienda, borradores de marketing).
6. **Mide e itera** (funnels, retención, DAU/MAU).
