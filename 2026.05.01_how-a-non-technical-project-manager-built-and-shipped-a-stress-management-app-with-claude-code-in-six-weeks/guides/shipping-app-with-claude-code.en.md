# Shipping a real app with Claude Code (case-study checklist)

Source: https://claude.com/blog/how-a-non-technical-project-manager-built-and-shipped-a-stress-management-app-with-claude-code-in-six-weeks

## What this guide covers
A practical checklist derived from the post, focusing on: prototype → specialization via subagents → production hardening → App Store release → measurement.

## Checklist
1. **Write a one-sentence goal** and list constraints (platform, devices, timeline).
2. **Prototype fast** with an end-to-end slice you can throw away.
3. **Assign specialist roles** (architecture, implementation, review, platform-specific experts).
4. **Harden for production** (crash reporting, analytics, screenshot-driven debugging).
5. **Prepare launch** (submission materials, store copy, marketing drafts).
6. **Measure and iterate** (funnels, retention, DAU/MAU).
