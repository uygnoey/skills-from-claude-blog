## この記事について
コード未経験のプロジェクトマネージャーであるKostiantyn Vlasenkoが、Claude Codeを中心に*iOS向けストレス管理アプリ「Respiro」*を開発し、6週間以内にApp Storeへ公開した事例を紹介します。出典: https://claude.com/blog/how-a-non-technical-project-manager-built-and-shipped-a-stress-management-app-with-claude-code-in-six-weeks

## どんなときに役立つか
- Claude Codeを主な開発環境として、実際のアプリを構想から公開まで進める流れを知りたいとき
- アーキテクチャ/実装/レビューなどの役割を分けたマルチエージェント（サブエージェント）運用を検討しているとき

## 主なポイント
- Built With Opus 4.6 Claude Code Hackathon（2026年2月）向けに約72時間で初期プロトタイプを作り、全体としては6週間以内にApp Store公開まで到達しました。
- TCAアーキテクト、Swift開発者、Metalスペシャリスト、コードレビュアーなど、15以上の専門サブエージェントを小さなチームのように調整したと述べています。
- コーディング以外にも、スクリーンショットを使ったデバッグ、App Store提出、マーケティング/グロースの作業にClaude Codeを活用しました。
