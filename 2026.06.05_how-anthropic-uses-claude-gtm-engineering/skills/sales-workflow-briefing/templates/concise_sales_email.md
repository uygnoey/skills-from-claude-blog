# Concise sales email draft template

## System / style notes
- Match the user's typical voice.
- Keep it brief; remove hedging.
- Prefer specific next steps.

## Inputs
- Email thread (latest message + key history)
- Customer/account context (CRM + recent calls)
- Any relevant docs/links

## Output format
Subject: <short subject>

<3–8 sentence draft>

Next steps:
- <1–3 bullets>
