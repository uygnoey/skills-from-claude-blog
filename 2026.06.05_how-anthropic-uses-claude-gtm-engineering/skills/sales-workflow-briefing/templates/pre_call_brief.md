# Pre-call brief template

## Inputs
- Today's calendar events (title, attendees, time, meeting link)
- Account/customer record (CRM)
- Recent emails with the customer
- Recent call notes/transcripts (if available)
- Relevant internal docs

## Output format
Account:
- Name:
- Stage / status:

What happened recently:
- <bullets>

Risks / blockers:
- <bullets>

Talking points:
- <bullets>

Questions to confirm:
- <bullets>
