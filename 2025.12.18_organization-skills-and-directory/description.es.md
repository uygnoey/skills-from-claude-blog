## ¿De qué trata este post?
Este post anuncia mejoras de las skills de Claude: gestión a nivel de organización para administradores, un directorio de skills creadas por partners y un estándar abierto pensado para que las skills sean portables entre plataformas de IA.

## ¿Cuándo es útil?
Es útil si administras Claude para un equipo/empresa, quieres desplegar skills de forma centralizada, o quieres descubrir skills de partners y entender cómo se despliegan en Claude Apps, Claude Code o mediante la API.

## Puntos clave
- Los administradores pueden aprovisionar skills para toda la organización desde la configuración de administración.
- Los usuarios pueden explorar y habilitar skills desde un directorio de partners.
- Las skills se publican como un estándar abierto, con la intención de funcionar entre herramientas y plataformas.
