## ¿De qué trata esta guía?
Esta guía resume las principales formas de desplegar y gestionar skills de Claude descritas en el post fuente: mediante Claude Apps, Claude Code o la API, con aprovisionamiento opcional a nivel de organización.

## Vías de despliegue
- **Claude Apps**: explora el directorio de skills y habilítalas en Settings > Capabilities > Skills.
- **Claude Code**: instala skills desde un directorio de plugins o incluye carpetas de skills en un repositorio.
- **API**: usa skills mediante el endpoint `/v1/skills` (consulta la documentación de la plataforma).

## Gestión a nivel de organización
- Los administradores (Team/Enterprise) pueden aprovisionar skills de forma central desde la configuración de administración.
- Algunas skills pueden requerir que estén habilitadas capacidades como Code Execution y File Creation.
