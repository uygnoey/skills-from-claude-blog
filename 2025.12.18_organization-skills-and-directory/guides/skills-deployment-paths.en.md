## What is this guide?
This guide summarizes the main ways to deploy and manage Claude skills described in the source post: via Claude Apps, Claude Code, or the API, with optional organization-wide provisioning.

## Deployment paths
- **Claude Apps**: browse the skills directory and enable skills from Settings > Capabilities > Skills.
- **Claude Code**: install skills from a plugin directory, or check skill folders into a repository.
- **API**: use skills via the `/v1/skills` endpoint (see platform documentation).

## Organization-wide management
- Admins (Team/Enterprise) can provision skills centrally from admin settings.
- Some skills require Code Execution and File Creation to be enabled.
