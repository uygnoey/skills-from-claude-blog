## What is this post?
This post announces improvements to Claude skills: organization-wide management for admins, a directory of partner-built skills, and an open standard designed to make skills portable across AI platforms.

## When is it useful?
Use it if you administer Claude for a team/enterprise, want to roll out skills centrally, or want to discover partner skills and understand how skills can be deployed in Claude Apps, Claude Code, or via the API.

## Key points
- Admins can provision skills organization-wide via admin settings.
- Users can browse and enable skills from a directory of partner-built skills.
- Skills are published as an open standard intended to work across tools and platforms.
