## 이 글이 뭔가요
이 글은 Claude 모델이 Microsoft Foundry에서 Azure 호스팅 형태로 정식 출시(GA)되었음을 알립니다.

## 언제 유용한가요
Azure 네이티브 환경에서 Claude를 운영하면서, 기존 Azure의 ID(인증), 네트워킹, 과금, 거버넌스 컨트롤을 그대로 활용하고 싶을 때 유용합니다.

## 핵심 포인트
- Messages API에서 Claude Opus 4.8, Claude Haiku 4.5를 사용할 수 있습니다.
- 주요 기능으로 프롬프트 캐싱과 확장 사고(extended thinking)를 강조합니다.
- 추론(inference) 처리 위치를 선택할 수 있으며, 데이터 레지던시 요구를 위한 미국(US) 데이터 존도 언급합니다.
- 청구서를 단일 인보이스로 통합할 수 있고, Microsoft Enterprise Agreement 대상 고객은 Azure 커밋먼트에서 사용량을 차감할 수 있습니다.
- “Azure 호스팅(hosted on Azure)”과 “Anthropic 호스팅(hosted on Anthropic, 기존 Foundry Preview)” 두 가지 옵션을 설명하며, 장기적으로 기능/모델 동등성(parity)을 목표로 한다고 말합니다.

## 번들 리소스
- 없음. (재사용 가능한 코드, hook, agent, output style, plugin 번들은 글에 포함되어 있지 않습니다.)
