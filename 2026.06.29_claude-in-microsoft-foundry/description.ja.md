## この記事について
この記事は、Claude モデルが Microsoft Foundry で Azure ホスティングとして一般提供（GA）になったことを発表するものです。

## どんなときに役立つか
Azure ネイティブな環境で Claude を運用し、既存の Azure の ID（認証）、ネットワーク、課金、ガバナンスのコントロールを活用したい場合に役立ちます。

## 主なポイント
- Messages API で Claude Opus 4.8 と Claude Haiku 4.5 が利用可能です。
- 主要機能として、プロンプトキャッシュと extended thinking（拡張思考）が挙げられています。
- 推論（inference）処理場所を選択でき、データレジデンシ要件向けに米国（US）データゾーンも言及されています。
- 請求を単一の請求書に統合でき、Microsoft Enterprise Agreement 対象顧客は Azure のコミットメントから利用分を充当できます。
- 「hosted on Azure」と「hosted on Anthropic（旧 Foundry Preview）」の 2 方式を説明し、将来的な機能・モデルの同等性（parity）を目指すとしています。
