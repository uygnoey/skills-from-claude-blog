## What is this post?
This post announces that Claude models are now generally available in Microsoft Foundry, hosted on Azure.

## When is it useful?
It’s useful if you want to run Claude in an Azure-native environment and use existing Azure identity, networking, billing, and governance controls.

## Key points
- Claude Opus 4.8 and Claude Haiku 4.5 are available via the Messages API.
- Core capabilities highlighted: prompt caching and extended thinking.
- You can choose where inference is processed, including a US data zone for data residency needs.
- Billing can be consolidated, and eligible Microsoft Enterprise Agreement customers can draw usage down from an Azure commitment.
- Two options are described: “hosted on Azure” and “hosted on Anthropic” (previously the Foundry Preview), with a goal of feature/model parity over time.
