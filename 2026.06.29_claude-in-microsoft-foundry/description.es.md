## ¿De qué trata este post?
Este post anuncia que los modelos Claude ya están disponibles de forma general (GA) en Microsoft Foundry, alojados en Azure.

## ¿Cuándo es útil?
Es útil si quieres ejecutar Claude en un entorno nativo de Azure y aprovechar la identidad, la red, la facturación y los controles de gobernanza que tu organización ya usa en Azure.

## Puntos clave
- Claude Opus 4.8 y Claude Haiku 4.5 están disponibles mediante la API de Messages.
- Se destacan dos capacidades principales: caché de prompts y pensamiento extendido (extended thinking).
- Puedes elegir dónde se procesa la inferencia, incluyendo una zona de datos en EE. UU. para requisitos de residencia de datos.
- La facturación puede consolidarse en una sola factura, y los clientes elegibles con Microsoft Enterprise Agreement pueden descontar el uso de un compromiso (commitment) de Azure.
- Se describen dos opciones: “hosted on Azure” y “hosted on Anthropic” (antes Foundry Preview), con el objetivo de lograr paridad de funciones y modelos con el tiempo.
