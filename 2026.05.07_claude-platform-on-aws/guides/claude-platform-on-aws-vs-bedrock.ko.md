# Claude Platform on AWS vs Claude on Amazon Bedrock

## 이 가이드가 다루는 내용

다음 두 옵션 중 무엇을 선택할지 빠르게 판단하기 위한 가이드입니다.

- **Claude Platform on AWS** (Anthropic이 운영하는 플랫폼을 AWS의 진입점으로 접근)
- **Claude on Amazon Bedrock** (AWS 경계 안에서 제공되는 AWS 관리형 서비스)

## 판단 체크리스트

### Claude Platform on AWS를 선택하기 좋은 경우

- AWS의 IAM/결제/CloudTrail을 통해 Anthropic의 네이티브 Claude Platform 경험을 쓰고 싶을 때
- 고객 데이터가 AWS 경계 밖에서 Anthropic에 의해 처리되는 것을 허용할 수 있을 때

### Claude on Amazon Bedrock을 선택하기 좋은 경우

- 데이터가 AWS 인프라 안에 머물러야 할 때
- 단일 AWS 서비스 안에서 AWS 관리형 기능과 더 넓은 모델 선택지를 선호할 때
