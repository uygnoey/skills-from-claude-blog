## What is this post?
This post announces **Claude Platform on AWS**, a way to access Anthropic’s first-party Claude Platform through AWS entry points (IAM, billing, and auditing).

## When is it useful?
- When your organization prefers AWS-native procurement, identity/access management, and audit trails.
- When you want to use Anthropic’s native Claude Platform experience without managing separate Anthropic credentials.

## Key points
- Access Claude Platform using existing **AWS credentials** and **IAM policies**.
- Usage is billed through **AWS consolidated billing**.
- Activity is logged to **AWS CloudTrail**.
- Important distinction: in this offering, **customer data is processed by Anthropic outside the AWS boundary**.

## Bundled resources
- A skill that helps teams evaluate and plan access controls, logging, and rollout considerations for “Claude Platform on AWS” vs “Claude on Amazon Bedrock”.
