## ¿De qué trata este post?
Este post presenta **Claude Platform on AWS**, una forma de acceder a la plataforma first-party de Anthropic (Claude Platform) mediante puntos de entrada de AWS (IAM, facturación y auditoría).

## ¿Cuándo es útil?
- Cuando tu organización prefiere unificar compras, identidad/acceso y auditoría con procesos nativos de AWS.
- Cuando quieres la experiencia nativa de Claude Platform sin gestionar credenciales separadas de Anthropic.

## Puntos clave
- Acceso a Claude Platform con **credenciales de AWS** y **políticas de IAM** existentes.
- El uso se cobra mediante la **facturación consolidada de AWS**.
- La actividad se registra en **AWS CloudTrail**.
- Distinción importante: en esta modalidad, **los datos del cliente son procesados por Anthropic fuera del perímetro de AWS**.

## Recursos incluidos
- Una skill para ayudar a equipos a evaluar y planificar controles de acceso, auditoría y despliegue para “Claude Platform on AWS” vs “Claude on Amazon Bedrock”.
