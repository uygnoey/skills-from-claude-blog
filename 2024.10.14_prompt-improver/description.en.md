## What is this post?
This post introduces a feature in the Anthropic Console that can automatically refine ("improve") your prompt and help you manage multi-shot examples in a structured way.

## When is it useful?
Use it when you want to make a prompt more reliable, clearer, and easier to evaluate, especially when you iterate on a prompt with examples.

## Key points
- The prompt improver applies multiple prompt-engineering techniques (e.g., adding structured reasoning sections, standardizing/enriching examples, rewriting for clarity, and adding prefills) to produce a revised prompt.
- You can provide feedback on what is or isn’t working and iterate.
- The Console provides example management and an evaluation workflow that can include an “ideal output” column for grading.
