## Instructions
Use this skill to guide prompt iteration when working in the Anthropic Console.

1. Start with a baseline prompt and define the desired output format.
2. Use the Console prompt improver to refine structure and clarity.
3. Add or edit multi-shot examples as input/output pairs to steer behavior.
4. Evaluate the prompt across scenarios and compare outputs to an ideal output when available.
5. Provide feedback on what isn’t working and repeat.

## Examples
- Improve a draft prompt for clarity and a stricter output format.
- Add a small set of examples, then evaluate the prompt against an ideal output and iterate.
