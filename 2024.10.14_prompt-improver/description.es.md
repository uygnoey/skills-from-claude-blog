## ¿De qué trata este post?
Este post presenta una función de la Anthropic Console que puede refinar ("improve") automáticamente tu prompt y ayudarte a gestionar ejemplos multi-shot de forma estructurada.

## ¿Cuándo es útil?
Es útil cuando quieres que un prompt sea más fiable, más claro y más fácil de evaluar, especialmente si iteras con ejemplos.

## Puntos clave
- El mejorador de prompts aplica varias técnicas de ingeniería de prompts (p. ej., añadir secciones de razonamiento estructurado, estandarizar/enriquecer ejemplos, reescribir para mayor claridad y añadir prefills) para producir un prompt revisado.
- Puedes dar feedback sobre lo que funciona o no funciona e iterar.
- La Console ofrece gestión de ejemplos y un flujo de evaluación que puede incluir una columna de “ideal output” para calificar.
