# Introducing the Max Plan

## ¿De qué trata este post?
Este post presenta el plan Max de Claude, que ofrece límites de uso más altos y acceso prioritario a nuevas funciones.

## ¿Cuándo es útil?
Es útil para decidir si necesitas límites más altos o un cambio de plan según tu carga diaria y plazos.

## Puntos clave
- Max está orientado a quienes colaboran con Claude de forma intensiva y necesitan más límites.
- Está pensado para conversaciones largas, documentos grandes, datos complejos y uso frecuente.
