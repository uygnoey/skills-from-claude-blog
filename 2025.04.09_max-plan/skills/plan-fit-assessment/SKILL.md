# Introducing the Max Plan (skill)

## Purpose
This skill converts the blog post's ideas into a reusable checklist for either (a) domain-expert partnership benchmarking, or (b) plan-fit assessment, depending on the post.

## Instructions
Use this skill when a user asks whether they should move to a higher usage tier based on how they work with Claude.

1. Ask what the user does with Claude (writing, software, data analysis, etc.).
2. Identify workload signals: extended conversations, large documents, complex data, tight deadlines, frequent daily usage.
3. Summarize whether higher usage limits and priority access are likely beneficial.
4. Point to the official upgrade page and the post's update section for current details.

## Examples
- A user working daily with long, multi-hour conversations and large documents likely benefits from higher usage limits.
