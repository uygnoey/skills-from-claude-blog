# Amazon Bedrock Model Distillation workflow (from the post)

The post describes Amazon Bedrock Model Distillation as automating:

1. Generating synthetic training data from the teacher model (Claude 3.5 Sonnet)
2. Training and evaluating the student model (Claude 3 Haiku)
3. Hosting the final distilled model for inference

Source: https://claude.com/blog/trainium2-and-distillation
