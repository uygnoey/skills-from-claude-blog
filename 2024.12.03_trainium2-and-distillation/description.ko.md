## 이 글이 뭔가요
이 글은 Amazon Bedrock에서 AWS Trainium2 기반의 Claude 3.5 Haiku 지연 시간 최적화(레이턴시 최적화) 추론을 미리보기로 제공하고, Amazon Bedrock 모델 디스틸레이션(distillation) 프리뷰 지원을 발표합니다.

## 언제 유용한가요
AWS에서 Claude 워크로드 배포 옵션을 선택할 때, 더 빠른 추론 옵션과 특정 작업에서 작은 모델의 성능을 끌어올리는 디스틸레이션 접근의 차이를 이해하는 데 유용합니다.

## 핵심 포인트
- Claude 3.5 Haiku는 Amazon Bedrock에서 레이턴시 최적화 추론(퍼블릭 프리뷰)을 지원하며, 최대 60% 더 빠른 추론 속도를 제공할 수 있다고 설명합니다.
- Amazon Bedrock 모델 디스틸레이션은 더 큰 “교사(teacher)” 모델의 지식을 더 작은 “학생(student)” 모델로 이전하기 위해, 합성 학습 데이터 생성 → 학습 및 평가 → 추론 호스팅의 흐름을 자동화합니다.
- Claude 3.5 Haiku 가격 인하도 함께 언급됩니다.
