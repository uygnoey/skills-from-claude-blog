## この記事について
この記事は、Amazon Bedrock 上で AWS Trainium2 に最適化した Claude 3.5 Haiku のレイテンシ最適化推論（パブリックプレビュー）と、Amazon Bedrock におけるモデル蒸留（distillation）のプレビュー対応を発表します。

## どんなときに役立つか
AWS で Claude のワークロードを展開する際に、より高速な推論オプションと、特定タスク向けに小型モデルを蒸留で強化するアプローチのトレードオフを理解したいときに役立ちます。

## 主なポイント
- Claude 3.5 Haiku は Amazon Bedrock でレイテンシ最適化推論（パブリックプレビュー）に対応し、最大 60% の推論高速化が可能だと述べています。
- Amazon Bedrock Model Distillation は、大型の「教師」モデルから小型の「生徒」モデルへ知識を移すために、合成データ生成 → 学習・評価 → 推論ホスティングを自動化します。
