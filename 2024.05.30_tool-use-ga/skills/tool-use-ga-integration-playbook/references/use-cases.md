# Tool use application patterns (from the post)

The post lists several common patterns that tool use enables:

- Extract structured data from unstructured text (e.g., names, dates, amounts from invoices).
- Convert natural-language requests into structured API calls (e.g., “cancel subscription”).
- Answer questions by searching databases or using web APIs.
- Automate simple tasks through software APIs.
- Orchestrate multiple fast Claude subagents for granular tasks (example: find an optimal meeting time based on attendee availability).

Source: https://claude.com/blog/tool-use-ga
