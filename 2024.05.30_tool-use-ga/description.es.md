## ¿De qué trata este post?
Este post anuncia la disponibilidad general del uso de herramientas (tool use) para la familia de modelos Claude 3, lo que permite a Claude llamar a herramientas y APIs externas en plataformas compatibles.

## ¿Cuándo es útil?
Es útil cuando estás decidiendo si incorporar llamadas a herramientas en un producto con Claude y quieres una visión general de las plataformas compatibles y los patrones de aplicación más comunes.

## Puntos clave
- El uso de herramientas está disponible de forma general para la familia Claude 3.
- Casos típicos: extracción estructurada desde texto no estructurado, convertir solicitudes en lenguaje natural a llamadas API, responder preguntas consultando sistemas externos y automatizar tareas mediante APIs de software.
- El post destaca funciones para desarrolladores como streaming y selección forzada de herramientas.
