## What is this post?
This post announces that tool use is generally available for the Claude 3 model family, enabling Claude to call external tools and APIs via supported platforms.

## When is it useful?
Use it when you are deciding whether to add tool calling to a Claude-powered product and want a high-level picture of supported platforms and common application patterns.

## Key points
- Tool use is generally available across the Claude 3 model family.
- Typical use cases include structured extraction from unstructured text, turning natural-language requests into API calls, answering questions by querying external systems, and automating tasks through software APIs.
- The post highlights developer-experience features like streaming and forced tool selection.
