## 이 글이 뭔가요
이 글은 Claude 3 모델 패밀리에서 도구 사용(tool use)이 정식 제공(GA)되어, 지원되는 플랫폼에서 외부 도구 및 API 호출을 통해 작업을 수행할 수 있게 되었음을 안내합니다.

## 언제 유용한가요
Claude 기반 제품에 도구 호출을 도입할지 판단할 때, 지원 플랫폼과 대표적인 활용 패턴을 빠르게 파악하는 데 유용합니다.

## 핵심 포인트
- Claude 3 모델 패밀리 전반에서 도구 사용이 GA로 제공됩니다.
- 비정형 텍스트에서 구조화된 데이터 추출, 자연어 요청의 API 호출 변환, 외부 시스템 질의 기반 Q&A, 소프트웨어 API를 통한 단순 업무 자동화 등 대표 사용 사례가 제시됩니다.
- 스트리밍 및 강제 도구 선택 같은 개발자 경험 관련 포인트를 강조합니다.
