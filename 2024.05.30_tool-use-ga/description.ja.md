## この記事について
この記事は、Claude 3 モデルファミリーでツール使用（tool use）が一般提供（GA）となり、対応プラットフォーム上で外部ツールや API を呼び出してタスクを実行できるようになったことを紹介します。

## どんなときに役立つか
Claude を使ったプロダクトにツール呼び出しを導入するか検討する際に、対応プラットフォームや代表的な活用パターンを俯瞰するのに役立ちます。

## 主なポイント
- Claude 3 モデルファミリー全体でツール使用が GA になりました。
- 非構造テキストからの構造化抽出、自然言語リクエストの API 呼び出し化、外部システム参照による Q&A、ソフトウェア API による簡易自動化などの代表例が挙げられています。
- ストリーミングや強制的なツール選択など、開発者体験に関する要素も強調されています。
