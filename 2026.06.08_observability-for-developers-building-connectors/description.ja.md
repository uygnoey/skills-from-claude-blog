# コネクターを構築する開発者のためのオブザーバビリティ

## この記事について
この記事では、Claude ディレクトリに公開されたコネクター向けの新しいオブザーバビリティ・ダッシュボードと、アプリ内から MCP サーバーをディレクトリへ直接申請できる機能を紹介しています。

## どんなときに役立つか
- 公開済みコネクターのオーナー／運用者として、Claude の各プロダクト面における採用状況・信頼性・性能を把握したいとき。
- MCP サーバーをコネクター・ディレクトリへ提出する手順をよりスムーズにしたいとき。

## 主なポイント
- ディレクトリ公開コネクターに、Claude 製品全体でのパフォーマンスを確認できるダッシュボードが追加されました。
- 採用状況（アクティブユーザー、総ツールコール数、ディレクトリ順位の推移）を追跡できます。
- 問題診断（ヘルススコア、エラー率、レイテンシ、ツール別エラー内訳）ができます。
- 製品別（例：Claude、Claude Code、Cowork など）に利用状況を比較できます。
- 利用には Team または Enterprise の Admin/Owner 権限（または Enterprise で Libraries 権限を持つカスタムロール）が必要です。
