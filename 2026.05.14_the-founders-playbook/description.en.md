# The founder's playbook: Building an AI-native startup

## What is this post?
This post is a stage-by-stage playbook for building an AI-native startup, describing how founders can use Claude across Idea, MVP, Launch, and Scale.

## When is it useful?
Useful when you are founding or operating an early-stage startup and want a practical operating system for using AI tools to move faster across the startup lifecycle.

## Key points
- Reframes startup execution into four stages (Idea, MVP, Launch, Scale) with AI-native workflows.
- Emphasizes founders shifting from individual contributor to orchestrator.
- Highlights a tool-choice matrix across Chat, Claude Cowork, and Claude Code per stage.
