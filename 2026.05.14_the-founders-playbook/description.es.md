# The founder's playbook: Building an AI-native startup

## ¿De qué trata este post?
Este post es un playbook por etapas para crear una startup nativa de IA, describiendo cómo usar Claude en Idea, MVP, Lanzamiento y Escala.

## ¿Cuándo es útil?
Útil si estás fundando u operando una startup en etapa temprana y quieres un sistema práctico para usar herramientas de IA a lo largo del ciclo de vida de la startup.

## Puntos clave
- Reframes startup execution into four stages (Idea, MVP, Launch, Scale) with AI-native workflows.
- Emphasizes founders shifting from individual contributor to orchestrator.
- Highlights a tool-choice matrix across Chat, Claude Cowork, and Claude Code per stage.
