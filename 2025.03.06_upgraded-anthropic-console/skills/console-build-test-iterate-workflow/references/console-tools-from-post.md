# Console tools and capabilities (from the post)

This reference reproduces what the March 6, 2025 launch post says about each piece of the upgraded Anthropic Console.

## Workbench — write prompts

> "Use the Workbench to structure your prompts effectively, incorporate examples, and integrate external tools, in an interactive environment for testing API calls."

The post positions the Workbench as the central authoring surface for prompts.

## Prompt generator — automatically generate prompts

> "Describe what you want to achieve, and Claude will use prompt engineering techniques such as chain-of-thought reasoning to create an effective, precise, and reliable prompt."

The generator is presented as the way to skip the blank-page problem.

## Evaluation tool — evaluate model responses

> "Evaluate your prompts against real-world scenarios with automatic test case generation and side-by-side output comparison. Easily run test suites, grade response quality, and make data-driven decisions about which prompts to deploy."

The post frames evaluation as the gate before deployment.

## Prompt improvement — improve prompts

> "Automatically refine prompts using advanced prompt engineering techniques. This is ideal for adapting prompts that were originally written for other AI models, as well as for optimizing hand-written prompts."

Prompt improvement is positioned for two specific entry points: cross-model migration and hand-written-prompt optimization.

## "Get Code" — production-ready export

> "After finalizing your prompt, you can click 'Get Code' for a production-ready API call you can immediately deploy."

This is the post's exit point from the iteration loop.

## Shareable prompts — collaborate across roles

> "Developers, domain experts, product managers, and QA specialists often need to collaborate on prompts to achieve optimal results. Previously, teams resorted to copying and pasting prompts between documents or chat applications, leading to version control issues and knowledge silos."
>
> "Shareable prompts in the Anthropic Console now provide a centralized way to develop, refine, and standardize prompts across your organization."

The post explicitly names developers, domain experts, product managers, and QA specialists as the target collaborators.

## Claude 3.7 Sonnet support and extended thinking budget

> "Claude 3.7 Sonnet, our latest and most intelligent model to date, can produce near-instant responses or extended, step-by-step thinking that is made visible to the user."
>
> "Simply specify the prompt will be used with extended thinking on, and Claude will generate the best responses possible. You can also use the Anthropic Console to control the budget for thinking by setting a max number of thinking tokens."

The post calls out two specific Console controls for extended thinking: telling Console that the prompt will be used with extended thinking on, and setting a max thinking-token budget.
