## ¿De qué trata este post?
Este post anuncia que Claude Enterprise está disponible en modalidad autoservicio y resume funciones empresariales y controles administrativos.

## ¿Cuándo es útil?
Es útil si estás evaluando o desplegando Claude Enterprise y necesitas un panorama rápido de la configuración, la seguridad y la gobernanza.

## Puntos clave
- Claude Enterprise can be purchased self-serve and set up quickly (workspace creation, SSO, inviting members).
- Identity and provisioning: SSO + domain capture, with SCIM for automated provisioning.
- Security visibility: audit logs and a Compliance API for monitoring usage and permissions.
- Admin controls: custom data retention policies, usage analytics for adoption/spend, and spend caps at org and user levels.
- Enterprise default: Anthropic does not train models on Claude Enterprise content by default.
