## What is this post?
This post shares practical guidance for adopting agentic coding across an engineering organization, based on common patterns observed across teams.

## When is it useful?
Use it when you want to move from small experiments to broader adoption, and need an approach for piloting, enablement, workflow changes, and success measurement.

## Key points
- Claude Enterprise can be purchased self-serve and set up quickly (workspace creation, SSO, inviting members).
- Identity and provisioning: SSO + domain capture, with SCIM for automated provisioning.
- Security visibility: audit logs and a Compliance API for monitoring usage and permissions.
- Admin controls: custom data retention policies, usage analytics for adoption/spend, and spend caps at org and user levels.
- Enterprise default: Anthropic does not train models on Claude Enterprise content by default.

## What is this post?
This post announces that Claude Enterprise is available for self-serve purchase and setup, and summarizes enterprise features and administrative controls.

## When is it useful?
Use it when you are evaluating or rolling out Claude Enterprise and need a quick overview of setup, security controls, and governance options.

## Key points
- Claude Enterprise can be purchased self-serve and set up quickly (workspace creation, SSO, inviting members).
- Identity and provisioning: SSO + domain capture, with SCIM for automated provisioning.
- Security visibility: audit logs and a Compliance API for monitoring usage and permissions.
- Admin controls: custom data retention policies, usage analytics for adoption/spend, and spend caps at org and user levels.
- Enterprise default: Anthropic does not train models on Claude Enterprise content by default.
