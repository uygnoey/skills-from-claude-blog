## Instructions
Use this skill as a rollout checklist when evaluating or deploying Claude Enterprise.

1. **Purchase and initialize** your workspace using the self-serve flow.
2. **Configure identity and access**: enable SSO and domain capture; set up SCIM provisioning if needed.
3. **Set governance controls**: define data retention policies and establish spend caps at org and user levels.
4. **Establish visibility**: enable audit logs and integrate with the Compliance API where required by your security/compliance program.
5. **Track adoption and spend** using usage analytics; adjust permissions and caps as needed.
