# Claude can now search the web

## What is this post?

Anthropic's March 20, 2025 announcement introduces web search inside Claude. With it, Claude can fetch up-to-date information from the internet and reply with cited responses, instead of relying only on training data. The post (with a May 27, 2025 update) confirms web search is available globally on all Claude plans, and walks through how four user types — sales, finance, research, and shopping — can benefit. The feature is initially toggled on per user via profile settings and used with Claude 3.7 Sonnet.

## When useful

- You are answering a question whose accuracy depends on recent events, prices, filings, or releases.
- You want Claude to cite the sources it relied on so you (or a downstream reviewer) can fact-check.
- You are building a workflow for sales, finance, research, or shopping that needs the latest data — the four use cases the post calls out by name.
- You want a conversational interface to web information instead of running a search engine yourself and stitching results together.

## Key points

- Claude can now search the web to deliver up-to-date, cited information in conversational responses across all plans.
- When Claude pulls in web information, it provides direct citations for fact-checking.
- The post lists four representative use cases: sales (account planning and trends), finance (market data, earnings, model assumptions), research (grant proposals, literature reviews, gap-finding), and shopping (price/feature/review comparisons).
- Initial rollout: feature preview for paid Claude users in the United States, with free-plan and additional-country support coming soon.
- May 27, 2025 update: web search is available globally on all Claude plans.
- To use it, toggle web search on in profile settings and start a conversation with Claude 3.7 Sonnet — Claude will search the web when applicable.
