# Claude が web 検索できるようになりました

## この記事について

Anthropic が 2025 年 3 月 20 日に公開した発表では、Claude に web 検索機能が導入されました。これにより Claude は、学習データだけに頼らず、インターネットから最新情報を取得し、引用付きで会話形式の回答を返すことができます。2025 年 5 月 27 日のアップデートにより、すべての Claude プランで全世界的に利用可能となり、記事はセールス / ファイナンス / リサーチ / ショッピングの 4 種類のユーザー向けの活用例を紹介しています。初期はプロフィール設定でトグルを有効にし、Claude 3.7 Sonnet と一緒に使用します。

## どんなときに役立つか

- 最近のイベント、価格、開示資料、リリースなど時点性の情報の正確さが重要な質問に答えるとき。
- Claude が参照したソースを引用込みで受け取り、自分自身もしくはレビュアーがファクトチェックできるようにしたいとき。
- セールス、ファイナンス、リサーチ、ショッピング — 記事が明示的に挙げた 4 領域で、最新データを使ったワークフローを構築したいとき。
- 検索エンジンを自分で叩いて結果をつなぎ合わせる代わりに、対話型インターフェースで Web 情報を得たいとき。

## 主なポイント

- Claude は web を検索して、最新情報と引用が含まれた会話形式の回答を、すべてのプランで提供できるようになりました。
- Web の情報を回答に組み込むときは、直接の引用を提示して事実確認を容易にします。
- 記事が挙げる 4 つの代表的な使い方:セールス(アカウントプランニングとトレンド分析)、ファイナンス(市場データ・決算・モデル前提)、リサーチ(助成提案・文献レビュー・ギャップ発見)、ショッピング(価格・機能・レビュー比較)。
- 初期ロールアウト:米国の有料 Claude ユーザー向けの feature preview。無料プランや他の国へのサポートも今後提供予定。
- 2025 年 5 月 27 日のアップデート:すべての Claude プランで全世界的に利用可能。
- 使い方:プロフィール設定で web search のトグルをオンにし、Claude 3.7 Sonnet と会話を開始 — 必要に応じて Claude が web 検索を行います。
