# Prompt patterns by role

These prompt patterns map directly to the four use cases the launch post calls out. They are illustrative starters meant to be edited with your own account names, tickers, topics, and products.

## Sales — pre-call account brief

```
Search the web for the latest initiatives, leadership changes, and publicly stated
pain points at <Account>. Restrict to sources from the past 90 days. Provide direct
citations.

For each finding, suggest one talk track I can use in an upcoming meeting with
<Buyer Title> at <Account>.
```

Why this fits the post: it directly executes the post's sales use case — "analyzing industry trends to learn key initiatives and pain points" to "transform account planning and drive higher win rates."

## Finance — earnings + consensus update

```
Search the web for <Company>'s most recent quarterly earnings report. Summarize:
- Reported revenue, operating margin, and any guidance changes.
- Current sell-side consensus on <metric> for the next quarter.
Provide citations to the filing(s) and to at least two reputable analyst notes.
```

Why this fits the post: the financial-analyst use case describes assessing "current market data, earnings reports, and industry trends to make better investment decisions and inform financial model assumptions."

## Research — literature review starter

```
Search the web for recent peer-reviewed papers on <topic> published in the last
2 years. Summarize:
- 5 most influential papers (with direct citations).
- 3 emerging trends across these papers.
- 2 gaps in the literature that a new study could address.
```

Why this fits the post: the researcher use case is "build stronger grant proposals and literature reviews by searching across primary sources on the web, spotting emerging trends and identifying gaps in the current literature."

## Shopping — comparison brief

```
Search the web to compare <Product A> vs <Product B>. Cover:
- Headline features and what is unique to each.
- Current price at major retailers.
- Highlights from recent reviews (last 6 months).
Cite retailer pages and trusted review sources directly.
```

Why this fits the post: the shopper use case is "compare product features, prices, and reviews across multiple sources to make more informed purchase decisions."
