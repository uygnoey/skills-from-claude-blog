# Claude ya puede buscar en la web

## ¿De qué trata este post?

El anuncio de Anthropic del 20 de marzo de 2025 introduce la búsqueda web dentro de Claude. Con ella, Claude puede traer información actualizada de internet y responder con citas, en lugar de depender únicamente de los datos de entrenamiento. La actualización del 27 de mayo de 2025 confirma que la búsqueda web está disponible globalmente en todos los planes de Claude, y el post recorre cómo cuatro tipos de usuarios — ventas, finanzas, investigación y compras — pueden aprovecharla. Inicialmente se activa por usuario en la configuración del perfil y se usa con Claude 3.7 Sonnet.

## ¿Cuándo es útil?

- Estás respondiendo una pregunta cuya exactitud depende de eventos, precios, presentaciones o lanzamientos recientes.
- Quieres que Claude cite las fuentes que utilizó para que tú (o quien revise) puedas verificar los hechos.
- Estás construyendo un flujo de trabajo para ventas, finanzas, investigación o compras que necesita los datos más recientes — los cuatro casos de uso que el post nombra explícitamente.
- Quieres una interfaz conversacional para acceder a información de la web, en vez de ejecutar un buscador y combinar resultados a mano.

## Puntos clave

- Claude puede buscar ahora en la web para entregar información actualizada y con citas en respuestas conversacionales en todos los planes.
- Cuando Claude incorpora información de la web, ofrece citas directas para verificación.
- El post enumera cuatro casos de uso representativos: ventas (planificación de cuentas y tendencias), finanzas (datos de mercado, earnings, supuestos del modelo), investigación (propuestas de subvención, revisiones de literatura, detección de huecos) y compras (comparación de precios, características y reseñas).
- Despliegue inicial: feature preview para usuarios de pago de Claude en Estados Unidos, con soporte para el plan gratuito y más países próximamente.
- Actualización del 27 de mayo de 2025: la búsqueda web está disponible globalmente en todos los planes de Claude.
- Para usarla: activar el toggle de búsqueda web en la configuración del perfil e iniciar una conversación con Claude 3.7 Sonnet — Claude buscará en la web cuando corresponda.
