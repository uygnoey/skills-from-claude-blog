# Claude가 이제 웹을 검색할 수 있습니다

## 이 글이 뭔가요

Anthropic이 2025년 3월 20일에 공개한 발표로, Claude 안에 웹 검색 기능을 도입한다는 내용입니다. 이 기능을 켜면 Claude는 학습 데이터에만 의존하지 않고 인터넷에서 최신 정보를 가져와 인용을 포함한 응답을 합니다. 2025년 5월 27일 업데이트에 따라 모든 Claude 플랜에서 전 세계적으로 사용할 수 있게 됐고, 글은 sales / finance / research / shopping 네 가지 사용자 유형의 활용 방식을 소개합니다. 처음에는 프로필 설정에서 토글을 켜고 Claude 3.7 Sonnet과 함께 사용합니다.

## 언제 유용한가요

- 최근의 이벤트, 가격, 공시, 출시 등 시점성 정보의 정확성이 중요한 질문에 답할 때.
- Claude가 참조한 출처를 인용까지 함께 받아 본인이나 검토자가 사실 확인을 할 수 있어야 할 때.
- sales, finance, research, shopping — 글이 직접 거론한 네 가지 영역에서 최신 데이터를 활용한 워크플로우를 만들 때.
- 검색 엔진을 직접 돌려 결과를 짜깁기하는 대신, 대화형 인터페이스로 웹 정보를 얻고 싶을 때.

## 핵심 포인트

- Claude는 이제 웹을 검색해 최신 정보와 인용이 포함된 대화형 응답을 모든 플랜에서 제공합니다.
- 웹 정보를 응답에 반영할 때 직접 인용을 함께 보여 사실 확인을 쉽게 합니다.
- 글이 명시한 4개의 대표 사용 사례: sales(계정 플래닝·트렌드 분석), finance(시장 데이터·실적·모델 가정), research(연구 제안·문헌 리뷰·갭 발굴), shopping(가격·기능·리뷰 비교).
- 초기 배포: 미국의 유료 Claude 사용자 대상 feature preview, 무료 플랜과 추가 국가는 곧 지원 예정.
- 2025년 5월 27일 업데이트: 모든 Claude 플랜에서 전 세계적으로 사용 가능.
- 사용 방법: 프로필 설정에서 web search 토글을 켜고 Claude 3.7 Sonnet과 대화 시작 — 필요할 때 Claude가 자동으로 웹을 검색합니다.
