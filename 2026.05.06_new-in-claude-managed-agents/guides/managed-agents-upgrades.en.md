# Managed Agents upgrades: dreaming, outcomes, orchestration, webhooks

## What is this guide?
A concise guide to the four capabilities described in the post and how they fit together.

## When is it useful?
Use it when you are deciding whether to adopt dreaming, outcomes, or multiagent orchestration in a Managed Agent.

## Key points
- **Memory vs dreaming**: memory captures learnings during work; dreaming refines memory between sessions.
- **Outcomes**: a rubric plus a grader that evaluates output in its own context window.
- **Multiagent orchestration**: a lead agent delegates to specialists who run in parallel on a shared filesystem.
- **Webhooks**: notify when outcomes evaluation completes.
