# Managed Agents の更新: dreaming、outcomes、オーケストレーション、Webhook

## このガイドについて
投稿で紹介された4つの機能と、その関係を短く整理したガイドです。

## どんなときに役立つか
Managed Agent に dreaming、outcomes、マルチエージェント・オーケストレーションを導入するか検討するときに役立ちます。

## 主なポイント
- **メモリ vs dreaming**: メモリは作業中の学習を保存し、dreaming はセッション間でメモリを洗練します。
- **Outcomes**: ルーブリックとグレーダーの組み合わせで、グレーダーが独立したコンテキストで評価します。
- **マルチエージェント・オーケストレーション**: リードエージェントが専門エージェントに並列で委任し、共有ファイルシステム上で協働します。
- **Webhook**: outcomes の評価完了を通知します。
