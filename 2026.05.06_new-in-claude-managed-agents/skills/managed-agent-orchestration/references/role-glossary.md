# Managed Agents role glossary

This glossary summarizes the roles described in the post.

- **Lead agent**: Breaks a complex job into sub-tasks, delegates to specialists, and checks in during execution to steer and integrate results.
- **Specialist agent**: A narrowly scoped agent with its own model, prompt, and tools; runs in parallel on a shared filesystem and contributes results back to the lead agent.
- **Grader**: A separate evaluator used by outcomes; it grades an output against a rubric in its own context window and indicates what changes are needed.
