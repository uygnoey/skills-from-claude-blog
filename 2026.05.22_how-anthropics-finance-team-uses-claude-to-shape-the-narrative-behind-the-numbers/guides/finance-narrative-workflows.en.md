# Finance narrative workflows (board deck + monthly review)

## Board deck (quarterly) checklist
1. Attach the current board deck.
2. Attach the latest source-of-truth numbers (model tables or approved exports).
3. Reconcile all numbers and claims to the source of truth.
4. Review the narrative like a board member:
   - contradictions
   - missing context/definitions
   - unclear baselines or time periods
5. Apply edits.
6. Repeat whenever numbers refresh.

## Monthly financial review checklist
1. Open the monthly review doc (tab per month).
2. Ensure it is structured as variance analysis versus forecast.
3. Paste/drop the relevant financial table from the model.
4. Link supporting context.
5. Draft the first pass in the established voice, then edit.
6. Preserve voice and conventions month over month.

## Project memory hygiene
- Keep separate projects for separate audiences (e.g., board deck vs. monthly review).
- Save important docs and decisions to project memory.
- For long cross-functional threads, extract the conclusion and reasoning and store it for later.
