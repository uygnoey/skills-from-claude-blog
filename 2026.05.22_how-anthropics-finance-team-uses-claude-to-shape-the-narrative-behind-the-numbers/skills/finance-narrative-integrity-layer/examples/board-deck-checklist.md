# Board deck workflow checklist

1) Attach the current board deck.
2) Attach the latest source-of-truth tables/exports.
3) Run reconciliation: [../templates/reconcile-claims.md](../templates/reconcile-claims.md)
4) Run board-reader review: [../templates/board-reader-review.md](../templates/board-reader-review.md)
5) Apply edits in the deck.
6) Re-run steps 3–4 whenever numbers refresh.
