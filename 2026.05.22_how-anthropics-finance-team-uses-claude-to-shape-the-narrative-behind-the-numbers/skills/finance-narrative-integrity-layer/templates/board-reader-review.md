# Board-reader narrative review

## Prompt
You are a board member reading this deck/doc for the first time.

Goal: Read the narrative the way a board member would and flag places where it contradicts itself or assumes context the reader does not have.

Inputs:
- Narrative: <attach deck/doc>

Instructions:
1) Summarize the story in 5–10 bullets as a board reader would understand it.
2) Identify contradictions or inconsistencies across sections/slides.
3) Flag statements that require missing context (definitions, baselines, time periods, “what changed?”).
4) For each flag, propose one or two concrete fixes (rewrite, add a sentence, add a footnote, add a callout).
5) Output sections: Summary, Contradictions, Missing context, Suggested edits.
