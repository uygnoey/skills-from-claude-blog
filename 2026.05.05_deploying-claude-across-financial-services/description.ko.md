# 금융 서비스 전반에 Claude 배포하기

## 이 글이 뭔가요

Anthropic이 2026년 5월 5일 발표한 금융 서비스용 Claude 배포 가이드입니다. 은행·자산운용사·보험사가 리서치, 딜, 언더라이팅, 클레임, 모델 리뷰, 월말 마감 등 여러 영역에 Claude를 어떻게 운영하는지 정리하고, Claude 제품군 전체(Chat, Cowork, Code, Microsoft 365 add-in, Claude Platform, Claude Managed Agents)와 사전 제작된 10개 금융 에이전트 템플릿, 3단계 도입 플레이북을 다루는 실무 가이드를 안내합니다.

## 언제 유용한가요

- 금융 서비스 기업의 AI 리더가 워크플로우별로 어떤 Claude 제품을 매칭할지 결정해야 할 때.
- 규제 환경에서 단계적 배포·거버넌스·고객 사례까지 함께 챙긴 Claude 도입 계획을 정리할 때.
- 사전 제작 10개 에이전트 템플릿(피치빌더, 미팅 준비, 어닝스 리뷰, 모델 빌더, 마켓 리서처, KYC 스크리너, 밸류에이션 리뷰, 총계정원장 정합화, 월말 마감, 재무제표 감사)을 기존 업무 체계에 맞춰 배치할 때.

## 핵심 포인트

- 대부분의 금융 서비스 기업은 이미 둘 이상의 Claude 제품을 병행 운영하고 있고, 가이드는 Chat / Cowork / Code / Microsoft 365 add-in / Claude Platform / Claude Managed Agents에 대한 product matrix를 명시적으로 제공합니다.
- 10개 사전 제작 에이전트 템플릿은 Claude Cowork·Claude Code의 플러그인 형태와 Claude Managed Agents용 cookbook 형태로 배포되며, 리서치/클라이언트 커버리지, 크레딧·리스크·컴플라이언스, 파이낸스·오퍼레이션 영역을 모두 포함합니다.
- 가이드는 AIG, Commonwealth Bank of Australia, IG Group, Moody's의 고객 사례를 함께 묶어둡니다.
- foundation → pilot → scale의 3단계 도입 플레이북이 단계별로 해야 할 일과 팁을 정리합니다.
- 활용 사례는 투자은행·사모펀드, 자산·자문, 리테일·상업 은행 같은 섹터별로 구조화되어 있습니다.
