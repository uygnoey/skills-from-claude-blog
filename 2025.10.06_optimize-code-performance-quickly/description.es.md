## ¿De qué trata este post?
Este post explica un flujo de trabajo práctico para acelerar código: usar Claude (chat) para analizar funciones lentas rápidamente y Claude Code para optimizar el rendimiento a nivel de proyecto.

## ¿Cuándo es útil?
Es útil cuando aparecen cuellos de botella en producción o durante el desarrollo y necesitas diagnóstico rápido, ideas de optimización y cambios coordinados en varios archivos.

## Puntos clave
- Usa Claude (chat) para analizar complejidad y proponer reescrituras en funciones pequeñas y aisladas.
- Usa Claude Code cuando el problema abarca varios archivos o requiere refactors coordinados y pruebas.
- Un buen inicio es instalar Claude Code y ejecutarlo dentro de los directorios más críticos para el rendimiento.
