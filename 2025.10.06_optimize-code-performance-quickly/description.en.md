## What is this post?
This post explains a practical workflow for speeding up code by using Claude for quick analysis of slow functions and Claude Code for project-wide performance optimization.

## When is it useful?
Use it when performance bottlenecks appear in production or during development and you need fast diagnosis, optimization ideas, and coordinated changes across multiple files.

## Key points
- Use Claude (chat) for quick complexity analysis and rewrite suggestions on small, isolated functions.
- Use Claude Code when performance issues span multiple files or require coordinated refactors and testing.
- A simple starting point is installing Claude Code and running it inside the most performance-critical directories.
