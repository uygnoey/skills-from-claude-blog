## 이 글이 뭔가요
이 글은 느린 함수는 Claude(채팅)로 빠르게 원인을 파악하고, 프로젝트 전반의 성능 이슈는 Claude Code로 분석·개선하는 실전 워크플로를 소개합니다.

## 언제 유용한가요
프로덕션이나 개발 과정에서 성능 병목이 나타났을 때, 빠른 진단과 최적화 아이디어, 그리고 여러 파일에 걸친 변경과 테스트까지 함께 진행해야 할 때 유용합니다.

## 핵심 포인트
- 작은 단일 함수는 Claude(채팅)로 복잡도/병목을 빠르게 분석하고 개선안을 얻습니다.
- 여러 파일에 걸친 성능 저하나 리팩터링/테스트가 필요한 경우 Claude Code를 사용합니다.
- Claude Code를 설치한 뒤 성능이 중요한 디렉터리에서 실행하는 것부터 시작할 수 있습니다.
