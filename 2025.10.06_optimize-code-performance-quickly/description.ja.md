## この記事について
この記事は、遅い関数を Claude（チャット）で素早く分析し、プロジェクト全体の性能最適化は Claude Code で進める実践的なワークフローを紹介します。

## どんなときに役立つか
本番環境や開発中に性能ボトルネックが現れ、迅速な原因特定・最適化アイデア・複数ファイルにまたがる変更とテストが必要なときに役立ちます。

## 主なポイント
- 小さく独立した関数は Claude（チャット）で計算量やボトルネックを素早く分析し、改善案を得ます。
- 複数ファイルにまたがる問題や協調的なリファクタリング／テストが必要な場合は Claude Code を使います。
- Claude Code をインストールし、性能が重要なディレクトリで実行するところから始められます。
