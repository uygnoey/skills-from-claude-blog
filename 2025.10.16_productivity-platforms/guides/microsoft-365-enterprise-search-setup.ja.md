# Claude + Microsoft 365: セットアップチェックリスト（記事ベース）

## このガイドの内容
記事で説明されている流れ（管理者が先に有効化・構成 → エンドユーザーが認証）に基づく簡易チェックリストです。

## チェックリスト
1. 管理者が Microsoft 365 コネクタを有効化します。
2. 管理者が含めるデータソース（SharePoint/OneDrive、Outlook、Teams）をキュレーションします。
3. 会社名でパーソナライズされた共有のエンタープライズ検索プロジェクトを作成/構成し、必要に応じてカスタムプロンプトを用意します。
4. エンドユーザーが Microsoft 365 アカウントで認証します。
