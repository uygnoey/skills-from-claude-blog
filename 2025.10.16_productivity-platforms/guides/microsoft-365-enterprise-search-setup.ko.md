# Claude + Microsoft 365: 설정 체크리스트(원문 기반)

## 이 가이드에서 다루는 내용
글에서 설명하는 흐름(관리자 선행 설정 → 사용자 인증)을 기준으로 한 간단한 체크리스트입니다.

## 체크리스트
1. 관리자가 Microsoft 365 커넥터를 활성화합니다.
2. 관리자가 포함할 데이터 소스(SharePoint/OneDrive, Outlook, Teams)를 선별합니다.
3. 회사 이름으로 개인화된 공유 엔터프라이즈 검색 프로젝트를 생성/구성하고, 필요 시 커스텀 프롬프트를 준비합니다.
4. 사용자는 자신의 Microsoft 365 계정으로 인증합니다.
