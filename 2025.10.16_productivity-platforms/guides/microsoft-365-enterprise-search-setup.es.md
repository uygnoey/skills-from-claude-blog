# Claude + Microsoft 365: checklist de configuración (según el post)

## Qué cubre esta guía
Un checklist ligero basado en el flujo descrito: primero el administrador habilita/configura, luego los usuarios finales se autentican.

## Checklist
1. El administrador habilita el conector de Microsoft 365.
2. El administrador selecciona qué fuentes se incluyen (SharePoint/OneDrive, Outlook, Teams).
3. Crear o configurar un proyecto compartido de búsqueda empresarial (con el nombre de la empresa) con prompts personalizados si aplica.
4. Los usuarios finales se autentican con sus cuentas de Microsoft 365.
