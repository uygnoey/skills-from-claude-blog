# Claude + Microsoft 365: setup checklist (from the post)

## What this guide covers
A lightweight checklist based on the post’s description: admin enables/configures first, then end users authenticate.

## Checklist
1. Admin enables the Microsoft 365 connector.
2. Admin curates which sources should be included (SharePoint/OneDrive, Outlook, Teams).
3. Create or configure a shared enterprise-search project (company-named) with any custom prompts.
4. End users authenticate with their Microsoft 365 accounts.
