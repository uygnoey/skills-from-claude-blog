# Example: Remote work policy report request

Please answer: What is our current remote work policy, and how has it evolved in the last 12 months?

Search across:
- HR policy documents in SharePoint/OneDrive
- Relevant Outlook threads discussing policy changes
- Any Teams channel guidance or meeting summaries where exceptions were discussed

Deliver a report with:
- Policy summary
- Exceptions and approval process
- Notable changes (with dates)
- Sources for each section (doc/email/thread identifiers)
