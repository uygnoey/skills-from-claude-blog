## この記事について
この記事は、Claude の Microsoft 365 連携と、接続された業務ツール全体に対するエンタープライズ検索（組織横断検索）の考え方を紹介します。

## どんなときに役立つか
業務コンテキストが Microsoft 365（ドキュメント、メール、カレンダー、チャット）に分散しており、ファイルを手動でアップロードせずに Claude に質問応答やレポート作成をさせたいときに役立ちます。

## 主なポイント
- Claude は MCP コネクタ経由で Microsoft 365（SharePoint/OneDrive、Outlook、Teams）に接続できます。
- エンタープライズ検索は、会社名でパーソナライズされた共有プロジェクトとして説明され、データソースのキュレーションとカスタムプロンプトを含みます。
- セットアップは管理者主導です。管理者がコネクタ/プロジェクトを有効化・構成し、その後ユーザーが認証して共有アクセスの恩恵を受けます。
