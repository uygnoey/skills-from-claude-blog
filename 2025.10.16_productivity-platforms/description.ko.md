## 이 글이 뭔가요
이 글은 Claude가 Microsoft 365와 연동되고, 연결된 업무 도구 전반에서 기업용(조직 단위) 검색을 제공한다는 내용을 소개합니다.

## 언제 유용한가요
업무 맥락이 Microsoft 365(문서, 이메일, 캘린더, 채팅)에 흩어져 있고, 파일을 직접 업로드하지 않아도 Claude가 질문에 답하거나 보고서를 작성하게 하고 싶을 때 유용합니다.

## 핵심 포인트
- Claude는 MCP 커넥터를 통해 Microsoft 365(SharePoint/OneDrive, Outlook, Teams)와 연결될 수 있습니다.
- 엔터프라이즈 검색은 회사 이름으로 개인화된 공유 프로젝트 형태로 설명되며, 데이터 소스 선별과 커스텀 프롬프트가 포함됩니다.
- 설정은 관리자 주도로 진행됩니다. 관리자가 커넥터/프로젝트를 활성화·구성하면, 이후 사용자는 인증만으로 조직 전체의 공유 접근 혜택을 받습니다.
