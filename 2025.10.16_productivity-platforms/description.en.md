## What is this post?
This post introduces Claude’s integrations with Microsoft 365 and the idea of enterprise-wide search across connected work tools.

## When is it useful?
Use it when your work context lives in Microsoft 365 (documents, email, calendar, chats) and you want Claude to answer questions or produce reports without manually uploading files.

## Key points
- Claude can connect to Microsoft 365 (SharePoint/OneDrive, Outlook, Teams) via an MCP connector.
- Enterprise search is presented as a shared, company-named project with curated data sources and custom prompts.
- Setup is admin-led: an administrator enables and configures the connector/project, then end users authenticate and benefit from shared access.

## Bundled resources
- A reusable skill bundle: `skills/microsoft-365-enterprise-search/` (instructions + templates + examples).
- A setup guide in four languages: `guides/microsoft-365-enterprise-search-setup.*.md`.
