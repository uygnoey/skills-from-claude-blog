# Skill types (from the post)

## Capability uplift

A capability uplift skill helps Claude do something the base model either can’t do or can’t do consistently.

The post cites document creation skills as examples, noting they encode techniques and patterns that can produce better output than prompting alone.

## Encoded preference

An encoded preference skill documents workflows where Claude can already do each piece, but the skill sequences the work according to your team’s process.

The post gives examples like:

- An NDA review skill that checks against set criteria
- A weekly update skill that drafts updates with data from various MCPs
