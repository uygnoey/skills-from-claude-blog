## この記事について
この記事は、Claude Code の Code Review 機能を紹介します。PR（プルリクエスト）が作成されると複数のエージェントを自動で起動してバグや問題点を探し、PR に要約コメントとインラインコメントとして結果を投稿します。

## どんなときに役立つか
- 軽量なリンターや LLM のざっとした確認よりも、深く丁寧なレビューが必要なとき
- すべての PR に一貫したレビューを自動で付けつつ、最終的な承認判断は人に残したいとき

## 主なポイント
- すべての PR にエージェントチームが投入され、PR の規模や複雑さに応じてレビューの深さが調整されます。
- 結果は、高シグナルの概要コメント 1 件と、具体的な指摘に対するインラインコメントとして届きます。
- 深さを重視した設計のため、軽量な選択肢よりコストが高くなることがあります。
- この機能が PR を承認することはなく、マージ可否は人が決めます。
